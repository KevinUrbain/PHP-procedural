# Structure de contrôle

# Étape 4 — Les structures de contrôle

## 1. La théorie complète

### Les conditions

`if` / `else` / `elseif`

```php
$age = 20;

if ($age < 18) {
    echo "Mineur";
} elseif ($age >= 18 && $age < 65) {
    echo "Adulte actif";
} else {
    echo "Senior";
}
```

**Règles importantes :**

- On teste du cas **le plus spécifique** au plus général
- On utilise **toujours** `===` sauf raison explicite
- Les accolades `{}` sont obligatoires même pour une seule ligne — lisibilité et sécurité

```php
// ❌ Sans accolades — dangereux
if ($age >= 18)
    echo "Majeur";
    echo "Bienvenue"; // Cette ligne s'exécute TOUJOURS, peu importe la condition !

// ✅ Avec accolades — sûr
if ($age >= 18) {
    echo "Majeur";
    echo "Bienvenue";
}
```

### `match` — le `switch` moderne (PHP 8+)

Le `switch` existe depuis longtemps mais a des pièges :

```php
// ❌ switch — comparaison LAXISTE, risque de bugs
$role = "admin";

switch ($role) {
    case "admin":
        echo "Accès total";
        break;       // ← oublier break = bug silencieux
    case "editor":
        echo "Accès éditeur";
        break;
    default:
        echo "Accès refusé";
}
```

```php
// ✅ match — comparaison STRICTE, plus sûr et plus concis (PHP 8+)
$role = "admin";

$acces = match($role) {
    "admin"          => "Accès total",
    "editor"         => "Accès éditeur",
    "viewer"         => "Accès lecture",
    default          => "Accès refusé"
};

echo $acces;
```

**Différences `switch` vs `match` :**

|  | `switch` | `match` |
| --- | --- | --- |
| Comparaison | `==` laxiste | `===` stricte |
| `break` requis | ✅ Oui | ❌ Non |
| Retourne une valeur | ❌ Non | ✅ Oui |
| PHP minimum | PHP 4 | PHP 8.0 |
| Recommandé | ❌ | ✅ |

### Les boucles

### `while` — tant que la condition est vraie

```php
// Exécute le bloc TANT QUE la condition est vraie
// La condition est vérifiée AVANT chaque itération

$compteur = 1;

while ($compteur <= 5) {
    echo "Tour n°{$compteur}<br>";
    $compteur++;    // ← INDISPENSABLE sinon boucle infinie !
}
```

### **`do...while` — exécute AU MOINS une fois**

```php
// La condition est vérifiée APRÈS chaque itération
// Le bloc s'exécute toujours au moins une fois

$tentatives = 0;

do {
    echo "Tentative n°{$tentatives}<br>";
    $tentatives++;
} while ($tentatives < 3);
```

### **`for` — quand on connaît le nombre d'itérations**

```php
// for (initialisation ; condition ; incrément)
for ($i = 1; $i <= 5; $i++) {
    echo "Ligne {$i}<br>";
}

// Compte à rebours
for ($i = 5; $i >= 1; $i--) {
    echo "Compte : {$i}<br>";
}
```

### **`foreach` — pour parcourir un tableau**

```php
// Tableau indexé
$prenoms = ["Alice", "Bob", "Charlie"];

foreach ($prenoms as $prenom) {
    echo "{$prenom}<br>";
}

// Tableau associatif — clé => valeur
$employe = [
    "prenom" => "Jack",
    "age"    => 35,
    "poste"  => "Développeur"
];

foreach ($employe as $cle => $valeur) {
    echo "{$cle} : {$valeur}<br>";
}
```

### **`break` et `continue`**

```php
// break — stoppe la boucle complètement
for ($i = 1; $i <= 10; $i++) {
    if ($i === 5) {
        break;  // Sort de la boucle dès que $i vaut 5
    }
    echo "{$i} ";   // Affiche : 1 2 3 4
}

// continue — saute l'itération courante et passe à la suivante
for ($i = 1; $i <= 10; $i++) {
    if ($i % 2 === 0) {
        continue;   // Saute les nombres pairs
    }
    echo "{$i} ";   // Affiche : 1 3 5 7 9
}
```

## 2. Exemple concret et utile

Système de **gestion des accès** d'une application web :

```php
<?php
declare(strict_types=1);

/**
 * acces.php
 * Gestion des droits d'accès selon le rôle utilisateur
 */

// Données simulées (viendront d'une DB en réalité)
$utilisateurs = [
    ["prenom" => "Jack",  "role" => "admin",  "actif" => true,  "score" => 98],
    ["prenom" => "Alice", "role" => "editor", "actif" => true,  "score" => 74],
    ["prenom" => "Bob",   "role" => "viewer", "actif" => false, "score" => 55],
    ["prenom" => "Sara",  "role" => "admin",  "actif" => true,  "score" => 87],
];

echo "<h2>Liste des utilisateurs</h2>";
echo "<ul>";

foreach ($utilisateurs as $user) {

    // On saute les utilisateurs inactifs
    if (!$user['actif']) {
        continue;
    }

    // Définition du niveau d'accès avec match
    $acces = match($user['role']) {
        "admin"  => "Accès total",
        "editor" => "Accès éditeur",
        "viewer" => "Accès lecture",
        default  => "Accès refusé"
    };

    // Définition du badge selon le score
    if ($user['score'] >= 90) {
        $badge = "🥇 Expert";
    } elseif ($user['score'] >= 75) {
        $badge = "🥈 Confirmé";
    } else {
        $badge = "🥉 Junior";
    }

    echo "<li>{$user['prenom']} — {$acces} — {$badge}</li>";
}

echo "</ul>";

// Affichage d'un tableau de multiplication avec for
echo "<h2>Table de 7</h2>";
echo "<ul>";

for ($i = 1; $i <= 10; $i++) {
    $resultat = 7 * $i;
    echo "<li>7 × {$i} = {$resultat}</li>";
}

echo "</ul>";
```

### **3. Bonnes pratiques et erreurs à éviter**

```php
// ❌ Condition sans accolades
if ($actif)
    echo "Actif";

// ✅ Toujours les accolades
if ($actif) {
    echo "Actif";
}

// ❌ switch avec == et sans break
switch ($role) {
    case 1:        // compare avec ==, 1 == "1" == true !
        echo "A";
    case 2:        // s'exécute aussi si on oublie le break !
        echo "B";
}

// ✅ match strict et sans risque
$label = match($role) {
    1       => "A",
    2       => "B",
    default => "Inconnu"
};

// ❌ Boucle infinie — incrément oublié
while ($i < 10) {
    echo $i;
    // $i++ oublié → boucle infinie → crash serveur
}

// ✅ Toujours vérifier que la condition peut devenir fausse
while ($i < 10) {
    echo $i;
    $i++;
}

// ❌ Modifier un tableau pendant un foreach sans référence
foreach ($prix as $p) {
    $p = $p * 1.10;  // Ne modifie PAS le tableau original
}

// ✅ Passage par référence pour modifier le tableau
foreach ($prix as &$p) {
    $p = $p * 1.10;  // Modifie le tableau original
}
unset($p); // ← INDISPENSABLE après un foreach par référence
```

## 4. Ce que font les seniors 👨‍💻

Les seniors **anticipent les cas limites** et structurent leur logique pour qu'elle soit lisible même 6 mois plus tard :

```php
<?php
declare(strict_types=1);

/**
 * Retourne le libellé du rôle — logique centralisée dans une fonction
 * On verra les fonctions à l'étape suivante, aperçu ici
 */

$role          = $_GET['role'] ?? 'guest';
$estConnecte   = true;
$tentativesMax = 3;
$tentatives    = 0;

// Guard clause — on sort tôt si les conditions ne sont pas remplies
// Evite les if imbriqués illisibles
if (!$estConnecte) {
    echo "Non connecté. Redirection...";
    exit; // On stoppe tout
}

// match avec cas multiples sur la même ligne
$permissions = match(true) {
    $role === 'admin'              => ['lire', 'écrire', 'supprimer'],
    in_array($role, ['editor'])    => ['lire', 'écrire'],
    default                        => ['lire']
};

// Boucle avec limite de sécurité
while ($tentatives < $tentativesMax) {
    // Simule une opération qui peut échouer
    $succes = true; // En réalité : appel API, requête DB...

    if ($succes) {
        echo "Opération réussie au tour n°{$tentatives}";
        break;
    }

    $tentatives++;
}
```

## 🎯 Exercice 4

Tu dois créer un **système de bulletins scolaires** pour une école.

Crée un fichier `bulletins.php` qui :

1. Déclare un tableau `$eleves` contenant **5 élèves**, chacun avec :
    - `prenom`
    - `note` (float entre 0 et 20)
    - `actif` (bool — un élève inactif = absent)
2. Avec une boucle `foreach` :
    - **Saute** les élèves inactifs avec `continue`
    - Calcule la **mention** avec `if/elseif/else` :
        - `>= 16` → `"Très bien"`
        - `>= 14` → `"Bien"`
        - `>= 12` → `"Assez bien"`
        - `>= 10` → `"Passable"`
        - `< 10` → `"Insuffisant"`
    - Calcule la **moyenne générale** de tous les élèves actifs
3. Avec `match`, affiche le **statut de la classe** selon la moyenne :
    - `>= 15` → `"Classe excellente"`
    - `>= 12` → `"Classe satisfaisante"`
    - `>= 10` → `"Classe passable"`
    - `default` → `"Classe en difficulté"`
4. Avec une boucle `for`, génère la **table de correspondance** notes/mentions (de 0 à 20 par pas de 2)
5. Affiche tout dans une page HTML propre avec un tableau pour les élèves et un second tableau pour la table de correspondance

Correction :

```php
<?php
declare(strict_types=1);

/**
 * bulletins.php
 * Système de bulletins scolaires
 */

$eleves = [
    ['prenom' => 'Josh',    'note' => 5.3,  'actif' => true],
    ['prenom' => 'Martin',  'note' => 18.3, 'actif' => true],
    ['prenom' => 'Sabine',  'note' => 8.9,  'actif' => true],
    ['prenom' => 'Erica',   'note' => 13.8, 'actif' => true],
    ['prenom' => 'Gustave', 'note' => 10.2, 'actif' => false]
];

// Initialisation AVANT la boucle
$somme               = 0;
$compteur            = 0;
$moyenneElevesActifs = 0;
$resultats           = []; // On stocke les résultats pour les afficher proprement

foreach ($eleves as $eleve) {

    // Sauter les élèves inactifs
    if (!$eleve['actif']) {
        continue;
    }

    // Calcul de la mention
    if ($eleve['note'] >= 16) {
        $mention = "Très bien";
    } elseif ($eleve['note'] >= 14) {
        $mention = "Bien";
    } elseif ($eleve['note'] >= 12) {
        $mention = "Assez bien";
    } elseif ($eleve['note'] >= 10) {
        $mention = "Passable";
    } else {
        $mention = "Insuffisant";
    }

    // Accumulation
    $somme += $eleve['note'];
    $compteur++;

    // On stocke le résultat pour l'affichage
    $resultats[] = [
        'prenom'  => $eleve['prenom'],
        'note'    => $eleve['note'],
        'mention' => $mention
    ];
}

// Calcul de la moyenne APRÈS la boucle + protection division par zéro
$moyenneElevesActifs = $compteur > 0 ? $somme / $compteur : 0;

// Statut de la classe
$statutClasse = match(true) {
    $moyenneElevesActifs >= 15 => "Classe excellente",
    $moyenneElevesActifs >= 12 => "Classe satisfaisante",
    $moyenneElevesActifs >= 10 => "Classe passable",
    default                    => "Classe en difficulté"
};

// Table de correspondance notes/mentions
$correspondances = [];

for ($i = 0; $i <= 20; $i += 2) {
    if ($i >= 16) {
        $mentionCorrespondance = "Très bien";
    } elseif ($i >= 14) {
        $mentionCorrespondance = "Bien";
    } elseif ($i >= 12) {
        $mentionCorrespondance = "Assez bien";
    } elseif ($i >= 10) {
        $mentionCorrespondance = "Passable";
    } else {
        $mentionCorrespondance = "Insuffisant";
    }

    $correspondances[] = ['note' => $i, 'mention' => $mentionCorrespondance];
}
?>
<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Bulletins scolaires</title>
</head>
<body>

    <h1>Bulletins scolaires</h1>

    <!-- Tableau des élèves actifs -->
    <h2>Résultats des élèves présents</h2>
    <table border="1">
        <thead>
            <tr>
                <th scope="col">Prénom</th>
                <th scope="col">Note</th>
                <th scope="col">Mention</th>
            </tr>
        </thead>
        <tbody>
            <?php foreach ($resultats as $resultat) : ?>
            <tr>
                <th scope="row"><?= $resultat['prenom'] ?></th>
                <td><?= number_format($resultat['note'], 2, ',', ' ') ?> / 20</td>
                <td><?= $resultat['mention'] ?></td>
            </tr>
            <?php endforeach; ?>
        </tbody>
        <tfoot>
            <tr>
                <td colspan="2">
                    Moyenne générale (<?= $compteur ?> élève(s) actif(s))
                </td>
                <td>
                    <?= number_format($moyenneElevesActifs, 2, ',', ' ') ?> / 20
                    — <?= $statutClasse ?>
                </td>
            </tr>
        </tfoot>
    </table>

    <!-- Table de correspondance -->
    <h2>Table de correspondance notes / mentions</h2>
    <table border="1">
        <thead>
            <tr>
                <th scope="col">Note</th>
                <th scope="col">Mention</th>
            </tr>
        </thead>
        <tbody>
            <?php foreach ($correspondances as $ligne) : ?>
            <tr>
                <td><?= $ligne['note'] ?> / 20</td>
                <td><?= $ligne['mention'] ?></td>
            </tr>
            <?php endforeach; ?>
        </tbody>
    </table>

</body>
</html>
```
