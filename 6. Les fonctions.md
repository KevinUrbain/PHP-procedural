# Étape 5 — Les fonctions

## 1. La théorie complète

### C'est quoi une fonction ?

Une fonction c'est un **bloc de code nommé, réutilisable**. Au lieu de réécrire la même logique plusieurs fois, tu l'encapsules dans une fonction et tu l'appelles quand tu en as besoin.

```php
// Sans fonction — tu répètes la logique partout ❌
$note1 = 15.5;
if ($note1 >= 16) { $mention = "Très bien"; }
elseif ($note1 >= 14) { $mention = "Bien"; }
else { $mention = "Insuffisant"; }

$note2 = 8.3;
if ($note2 >= 16) { $mention = "Très bien"; }  // copier/coller = danger
elseif ($note2 >= 14) { $mention = "Bien"; }
else { $mention = "Insuffisant"; }

// Avec fonction — tu écris la logique UNE fois ✅
function getMention(float $note): string {
    if ($note >= 16)      return "Très bien";
    elseif ($note >= 14)  return "Bien";
    else                  return "Insuffisant";
}

echo getMention(15.5);  // "Bien"
echo getMention(8.3);   // "Insuffisant"
```

### **Anatomie d'une fonction**

```php
function nomDeLaFonction(type $parametre1, type $parametre2): typeRetour
{
    // Corps de la fonction
    return $valeur;
}
```

```php
//   ┌── mot clé
//   │        ┌── nom (camelCase)
//   │        │           ┌── paramètre typé
//   │        │           │              ┌── type de retour
function additionner(int $a, int $b): int
{
    return $a + $b;  // ← valeur retournée
}

$resultat = additionner(3, 5);  // 8
```

### **Les paramètres**

```php
// Paramètre simple
function direBonjour(string $prenom): string
{
    return "Bonjour {$prenom} !";
}

// Paramètre avec valeur par défaut
function direBonjour(string $prenom = "inconnu"): string
{
    return "Bonjour {$prenom} !";
}

echo direBonjour("Jack");   // "Bonjour Jack !"
echo direBonjour();         // "Bonjour inconnu !"
```

### **Le type de retour**

```php
// Retourne un string
function getNom(): string { return "Jack"; }

// Retourne un int
function getAge(): int { return 35; }

// Retourne un float
function getPrix(): float { return 19.99; }

// Retourne un bool
function estMajeur(int $age): bool { return $age >= 18; }

// Retourne un array
function getCoordonnees(): array { return ["x" => 10, "y" => 20]; }

// Ne retourne rien — void
function afficherMessage(string $msg): void { echo $msg; }

// Peut retourner null ou un string — nullable
function chercherUtilisateur(int $id): ?string
{
    // Simule une recherche
    if ($id === 1) return "Jack";
    return null;   // Pas trouvé
}
```

### Portée des variables — scope

C'est un concept **crucial** en PHP :

```php
$nom = "Jack";   // Variable GLOBALE

function direBonjour(): void
{
    // $nom n'existe PAS ici — PHP isole les fonctions
    echo $nom;   // ❌ Undefined variable
}

// Solution 1 — passer en paramètre ✅ (recommandé)
function direBonjour(string $nom): void
{
    echo $nom;   // ✅
}
direBonjour($nom);

// Solution 2 — mot clé global (déconseillé)
function direBonjour(): void
{
    global $nom;   // Importe la variable globale
    echo $nom;     // ✅ mais mauvaise pratique
}
```

### **Passage par valeur vs par référence**

```php
// Par VALEUR — la fonction reçoit une copie
function augmenterPrix(float $prix): float
{
    $prix += 10;    // Modifie la copie locale
    return $prix;
}

$prix = 100.0;
$nouveauPrix = augmenterPrix($prix);
echo $prix;        // 100 — inchangé
echo $nouveauPrix; // 110

// Par RÉFÉRENCE — la fonction modifie la variable originale
function augmenterPrixRef(float &$prix): void
{
    $prix += 10;   // Modifie la variable originale
}

$prix = 100.0;
augmenterPrixRef($prix);
echo $prix;   // 110 — modifié directement
```

### **Fonctions anonymes (closures)**

```php
// Fonction sans nom — assignée à une variable
$calculerTTC = function(float $prixHT, float $tva): float {
    return $prixHT * (1 + $tva);
};

echo $calculerTTC(100.0, 0.21);   // 121.0

// Très utilisées avec array_map, array_filter, usort...
$prix = [100.0, 200.0, 300.0];

$prixTTC = array_map(function(float $p): float {
    return $p * 1.21;
}, $prix);

// [121.0, 242.0, 363.0]
```

### Fonctions fléchées — arrow functions (PHP 7.4+)

Version courte d'une closure pour les cas simples :

```php
// Closure classique
$double = function(int $n): int { return $n * 2; };

// Arrow function — équivalent plus concis
$double = fn(int $n): int => $n * 2;

echo $double(5);   // 10

// Avec array_map
$prix    = [100.0, 200.0, 300.0];
$prixTTC = array_map(fn(float $p): float => $p * 1.21, $prix);
```

## 2. Exemple concret et utile

Un **module de calcul de paie** réutilisable — tu reconnais l'exercice 3, mais cette fois proprement structuré avec des fonctions :

```php
<?php
declare(strict_types=1);

/**
 * paie.php
 * Module de calcul de paie — fonctions réutilisables
 */

const TAUX_ONSS = 0.1307;
const TAUX_IPP  = 0.2650;

// ============================================================
// FONCTIONS
// ============================================================

/**
 * Calcule le montant des heures supplémentaires
 */
function calculerHeuresSup(int $heures, float $tauxHoraire): float
{
    return $heures * $tauxHoraire;
}

/**
 * Calcule le salaire brut total (mensuel + heures sup)
 */
function calculerBrutTotal(float $salaireMensuel, float $montantHeuresSup): float
{
    return $salaireMensuel + $montantHeuresSup;
}

/**
 * Calcule la cotisation ONSS
 */
function calculerONSS(float $brutTotal): float
{
    return $brutTotal * TAUX_ONSS;
}

/**
 * Calcule l'IPP sur le salaire après ONSS
 */
function calculerIPP(float $brutApresOnss): float
{
    return $brutApresOnss * TAUX_IPP;
}

/**
 * Calcule le salaire net final
 */
function calculerNetFinal(float $brutTotal): float
{
    $onss         = calculerONSS($brutTotal);
    $brutApresONSS = $brutTotal - $onss;
    $ipp          = calculerIPP($brutApresONSS);

    return $brutApresONSS - $ipp;
}

/**
 * Retourne le statut du salaire
 */
function getStatutSalaire(float $salaireNet): string
{
    return $salaireNet > 2500 ? "Salaire confortable" : "Salaire modeste";
}

// ============================================================
// UTILISATION
// ============================================================

$salaireMensuel = 3509.57;
$heuresSup      = 29;
$tauxHoraire    = 15.40;

$montantHeuresSup = calculerHeuresSup($heuresSup, $tauxHoraire);
$brutTotal        = calculerBrutTotal($salaireMensuel, $montantHeuresSup);
$salaireNet       = calculerNetFinal($brutTotal);
$statut           = getStatutSalaire($salaireNet);

echo "Brut total : "   . number_format($brutTotal, 2, ',', ' ')   . " €<br>";
echo "Salaire net : "  . number_format($salaireNet, 2, ',', ' ')  . " €<br>";
echo "Statut : "       . $statut                                   . "<br>";
```

## 3. Bonnes pratiques et erreurs à éviter

```php
// ❌ Fonction qui fait trop de choses
function traiterEmploye(array $employe): void
{
    // Calcule le salaire ET affiche ET envoie un email...
    // Une fonction = une responsabilité
}

// ✅ Une fonction = une responsabilité
function calculerNet(float $brut): float { ... }
function formaterSalaire(float $net): string { ... }
function afficherFiche(array $employe): void { ... }

// ❌ Pas de typage
function calculer($a, $b) { return $a + $b; }

// ✅ Toujours typer paramètres et retour
function calculer(float $a, float $b): float { return $a + $b; }

// ❌ Nom vague
function faire(array $data): array { ... }

// ✅ Nom expressif — verbe + complément
function filtrerProduitsActifs(array $produits): array { ... }
function calculerMoyenneNotes(array $notes): float { ... }
function getStatutCommande(string $statut): string { ... }
```

## 4. Ce que font les seniors 👨‍💻

Les seniors organisent leurs fonctions dans des **fichiers dédiés** et documentent avec PHPDoc :

```php
<?php
// helpers/paie.php — toutes les fonctions liées à la paie

declare(strict_types=1);

/**
 * Calcule le salaire net à partir du brut
 *
 * @param float $brutTotal    Salaire brut total mensuel
 * @param float $tauxOnss     Taux de cotisation ONSS
 * @param float $tauxIpp      Taux d'imposition IPP
 * @return float              Salaire net final
 */
function calculerSalaireNet(
    float $brutTotal,
    float $tauxOnss = TAUX_ONSS,
    float $tauxIpp  = TAUX_IPP
): float {
    $onss          = $brutTotal * $tauxOnss;
    $brutApresOnss = $brutTotal - $onss;
    $ipp           = $brutApresOnss * $tauxIpp;

    return $brutApresOnss - $ipp;
}
```

Et dans le fichier principal :

```php
<?php
// index.php
declare(strict_types=1);

require_once 'config/constants.php';
require_once 'helpers/paie.php';      // ← on importe les fonctions

$net = calculerSalaireNet(3509.57);   // disponible partout
```

## 🎯 Exercice 5

On reprend le système de **bulletins scolaires** de l'exercice 4 — mais cette fois on va **tout refactoriser avec des fonctions**.

Crée un fichier `bulletins.php` qui contient :

1. Une fonction `getMention(float $note): string` qui retourne la mention selon la note
2. Une fonction `filtrerElevesActifs(array $eleves): array` qui retourne uniquement les élèves actifs
3. Une fonction `calculerMoyenne(array $eleves): float` qui calcule la moyenne des notes des élèves reçus en paramètre — retourne `0.0` si le tableau est vide
4. Une fonction `getStatutClasse(float $moyenne): string` qui retourne le statut avec `match`
5. Une fonction `enrichirEleves(array $eleves): array` qui retourne le tableau des élèves enrichi avec leur mention — utilise `getMention()` à l'intérieur
6. Dans le code principal :
    - Déclare le tableau `$eleves` avec 5 élèves (mélange actif/inactif)
    - Utilise tes fonctions dans le bon ordre pour obtenir les résultats
    - Affiche tout dans une page HTML avec un tableau propre

Correction :

```php
<?php
declare(strict_types=1);

/**
 * bulletins.php
 * Système de bulletins scolaires — version avec fonctions
 */

// ============================================================
// DONNÉES
// ============================================================

$eleves = [
    ['prenom' => 'Josh',    'note' => 5.3,  'actif' => true],
    ['prenom' => 'Martin',  'note' => 18.3, 'actif' => true],
    ['prenom' => 'Sabine',  'note' => 8.9,  'actif' => true],
    ['prenom' => 'Erica',   'note' => 13.8, 'actif' => true],
    ['prenom' => 'Gustave', 'note' => 10.2, 'actif' => false]
];

// ============================================================
// FONCTIONS
// ============================================================

/**
 * Retourne la mention correspondant à une note sur 20
 *
 * @param float $note Note entre 0 et 20
 * @return string     Mention associée
 */
function getMention(float $note): string
{
    if ($note >= 16)      return "Très bien";
    elseif ($note >= 14)  return "Bien";
    elseif ($note >= 12)  return "Assez bien";
    elseif ($note >= 10)  return "Passable";
    else                  return "Insuffisant";
}

/**
 * Retourne uniquement les élèves actifs
 *
 * @param array $eleves Tableau complet des élèves
 * @return array        Tableau filtré des élèves actifs
 */
function filtrerElevesActifs(array $eleves): array
{
    $elevesActifs = [];

    foreach ($eleves as $eleve) {
        if ($eleve['actif']) {
            $elevesActifs[] = $eleve;
        }
    }

    return $elevesActifs;
}

/**
 * Calcule la moyenne des notes d'un tableau d'élèves
 *
 * @param array $eleves Tableau d'élèves avec clé 'note'
 * @return float        Moyenne des notes, 0.0 si tableau vide
 */
function calculerMoyenne(array $eleves): float
{
    if (count($eleves) === 0) {
        return 0.0;
    }

    $somme = 0.0;

    foreach ($eleves as $eleve) {
        $somme += $eleve['note'];
    }

    return $somme / count($eleves);
}

/**
 * Retourne le statut de la classe selon la moyenne
 *
 * @param float $moyenne Moyenne générale de la classe
 * @return string        Statut de la classe
 */
function getStatutClasse(float $moyenne): string
{
    return match(true) {
        $moyenne >= 15 => "Classe excellente",
        $moyenne >= 12 => "Classe satisfaisante",
        $moyenne >= 10 => "Classe passable",
        default        => "Classe en difficulté"
    };
}

/**
 * Enrichit chaque élève avec sa mention calculée
 *
 * @param array $eleves Tableau d'élèves avec clé 'note'
 * @return array        Tableau d'élèves avec clé 'mention' ajoutée
 */
function enrichirEleves(array $eleves): array
{
    $elevesEnrichis = [];

    foreach ($eleves as $eleve) {
        $eleve['mention'] = getMention($eleve['note']);
        $elevesEnrichis[] = $eleve;
    }

    return $elevesEnrichis;
}

// ============================================================

// ============================================================

$elevesActifs        = filtrerElevesActifs($eleves);
$elevesEnrichis      = enrichirEleves($elevesActifs);
$moyenneElevesActifs = calculerMoyenne($elevesEnrichis);
$statutClasse        = getStatutClasse($moyenneElevesActifs);
?>

            </tr>

        </tbody>
```

## Ce que tu dois retenir
```
┌─────────────────────────────────────────────┐
│  Une fonction = une responsabilité          │
│  Un nom = un verbe + un complément          │
│  Après un return dans un if → pas de else   │
│  On retourne directement, sans variable     │
│  intermédiaire inutile                      │
│  PHPDoc = @param + @return toujours         │
└─────────────────────────────────────────────┘
```

⚠️ Attention à la logique de la fonction `enrichirEleves()`

Mon premier code tenté :

```php
// ❌ Tu utilises $index mais tu fais $elevesEnrichis[] en même temps
// Ce qui crée une incohérence potentielle
foreach ($eleves as $index => $eleve) {
    $mentions = getMention($eleve['note']);
    $elevesEnrichis[] = $eleve;              // ajoute à l'index 0, 1, 2...
    $elevesEnrichis[$index]['mention'] = $mentions;  // utilise l'index original
}
```

Si les index du tableau d'entrée ne commencent pas à 0, ça peut créer des trous. La bonne approche :

```php
// ✅ On enrichit directement avant de stocker
function enrichirEleves(array $eleves): array
{
    $elevesEnrichis = [];

    foreach ($eleves as $eleve) {
        $eleve['mention'] = getMention($eleve['note']); // on ajoute la clé
        $elevesEnrichis[] = $eleve;                     // on stocke l'ensemble
    }

    return $elevesEnrichis;
}
```
