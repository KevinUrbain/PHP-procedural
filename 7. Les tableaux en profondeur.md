# Les tableaux en profondeur

# Étape 6 — Les tableaux en profondeur

## 1. La théorie complète

On connaît déjà les bases des tableaux. Ici on va voir les **fonctions natives PHP** qui permettent de manipuler les tableaux de façon professionnelle — sans écrire des boucles à la main.

### Les fonctions essentielles

### `count()` — compter les éléments

```php
$fruits = ["pomme", "banane", "cerise"];
echo count($fruits);   // 3

$eleves = [
    ["prenom" => "Jack"],
    ["prenom" => "Alice"],
];
echo count($eleves);   // 2 — compte les sous-tableaux, pas les clés
```

### **`in_array()` — vérifier si une valeur existe**

```php
$roles = ["admin", "editor", "viewer"];

var_dump(in_array("admin", $roles));    // bool(true)
var_dump(in_array("guest", $roles));    // bool(false)

// 3ème paramètre = true pour comparaison stricte (type + valeur)
$ids = [1, 2, 3];
var_dump(in_array("1", $ids));          // bool(true)  — laxiste
var_dump(in_array("1", $ids, true));    // bool(false) — strict
```

### **`array_keys()` et `array_values()` — extraire clés ou valeurs**

```php
$employe = [
    "prenom" => "Jack",
    "age"    => 35,
    "poste"  => "Développeur"
];

$cles   = array_keys($employe);
$valeurs = array_values($employe);

var_dump($cles);    // ["prenom", "age", "poste"]
var_dump($valeurs); // ["Jack", 35, "Développeur"]
```

### **`array_push()` et `array_pop()` — ajouter/retirer en fin**

```php
$panier = ["pomme", "banane"];

array_push($panier, "cerise");        // Ajoute à la fin
// Equivalent plus simple : $panier[] = "cerise";

$dernier = array_pop($panier);        // Retire et retourne le dernier
echo $dernier;   // "cerise"
var_dump($panier);   // ["pomme", "banane"]
```

### **`array_merge()` — fusionner des tableaux**

```php
$tableau1 = ["pomme", "banane"];
$tableau2 = ["cerise", "kiwi"];

$fusion = array_merge($tableau1, $tableau2);
// ["pomme", "banane", "cerise", "kiwi"]

// Avec tableaux associatifs — les clés identiques sont écrasées
$defauts  = ["couleur" => "bleu", "taille" => "M"];
$surcharge = ["couleur" => "rouge"];

$final = array_merge($defauts, $surcharge);
// ["couleur" => "rouge", "taille" => "M"]
```

### **`array_slice()` — extraire une portion**

```php
$nombres = [1, 2, 3, 4, 5, 6];

// array_slice(tableau, début, longueur)
$portion = array_slice($nombres, 1, 3);
var_dump($portion);   // [2, 3, 4]

// Utile pour la pagination
$tousLesArticles = [/* 100 articles */];
$page            = 2;
$parPage         = 10;
$articlesDuPage  = array_slice($tousLesArticles, ($page - 1) * $parPage, $parPage);
```

### **`array_search()` — trouver l'index d'une valeur**

```php
$fruits = ["pomme", "banane", "cerise"];

$index = array_search("banane", $fruits);
echo $index;   // 1

// Retourne false si non trouvé
$index = array_search("kiwi", $fruits);
var_dump($index);   // bool(false)
```

### **`array_unique()` — supprimer les doublons**

```php
$tags = ["php", "web", "php", "css", "web"];

$unique = array_unique($tags);
var_dump($unique);   // ["php", "web", "css"]
```

### **`sort()` et `arsort()` — trier**

```php
// sort() — tri croissant, réindexe
$notes = [15.5, 8.3, 18.0, 12.5];
sort($notes);
var_dump($notes);   // [8.3, 12.5, 15.5, 18.0]

// rsort() — tri décroissant
rsort($notes);
var_dump($notes);   // [18.0, 15.5, 12.5, 8.3]

// asort() — tri croissant, CONSERVE les clés associatives
$prix = ["laptop" => 899, "souris" => 29, "clavier" => 79];
asort($prix);
// ["souris" => 29, "clavier" => 79, "laptop" => 899]

// ksort() — tri par CLÉ
ksort($prix);
// ["clavier" => 79, "laptop" => 899, "souris" => 29]
```

### Les trois fonctions de transformation — le cœur du PHP moderne

### `array_map()` — transformer chaque élément

Applique une fonction à **chaque élément** et retourne un nouveau tableau :

```php
$prix = [100.0, 200.0, 300.0];

// Ancienne façon — boucle manuelle
$prixTTC = [];
foreach ($prix as $p) {
    $prixTTC[] = $p * 1.21;
}

// Moderne — array_map
$prixTTC = array_map(fn(float $p): float => $p * 1.21, $prix);
// [121.0, 242.0, 363.0]

// Avec une fonction nommée
function appliquerTVA(float $prix): float {
    return $prix * 1.21;
}
$prixTTC = array_map('appliquerTVA', $prix);
```

### `array_filter()` — filtrer les éléments

Garde uniquement les éléments pour lesquels la fonction retourne `true` :

```php
$produits = [
    ["nom" => "Laptop",  "stock" => 5],
    ["nom" => "Souris",  "stock" => 0],
    ["nom" => "Clavier", "stock" => 3],
];

// Ancienne façon — boucle manuelle
$enStock = [];
foreach ($produits as $p) {
    if ($p['stock'] > 0) $enStock[] = $p;
}

// Moderne — array_filter
$enStock = array_filter(
    $produits,
    fn(array $p): bool => $p['stock'] > 0
);

// Attention — array_filter CONSERVE les index originaux
// Pour réindexer :
$enStock = array_values(array_filter(
    $produits,
    fn(array $p): bool => $p['stock'] > 0
));
```

### `array_reduce()` — réduire un tableau à une seule valeur

Très puissant pour calculer des totaux, construire des structures :

```php
$produits = [
    ["nom" => "Laptop",  "prix" => 899.0, "stock" => 2],
    ["nom" => "Souris",  "prix" => 29.0,  "stock" => 5],
    ["nom" => "Clavier", "prix" => 79.0,  "stock" => 3],
];

// Calcul de la valeur totale du stock
// array_reduce(tableau, fonction(accumulateur, element), valeur_initiale)
$valeurTotale = array_reduce(
    $produits,
    fn(float $total, array $p): float => $total + ($p['prix'] * $p['stock']),
    0.0   // valeur initiale de l'accumulateur
);

echo $valeurTotale;   // (899*2) + (29*5) + (79*3) = 2279.0
```

### **`usort()` — tri personnalisé**

```php
$eleves = [
    ["prenom" => "Josh",   "note" => 5.3],
    ["prenom" => "Martin", "note" => 18.3],
    ["prenom" => "Sabine", "note" => 8.9],
];

// Tri par note croissante
usort($eleves, fn(array $a, array $b): int => $a['note'] <=> $b['note']);

// Tri par note décroissante
usort($eleves, fn(array $a, array $b): int => $b['note'] <=> $a['note']);
```

## **3. Bonnes pratiques et erreurs à éviter**

```php
// ❌ Modifier le tableau pendant un foreach sans référence
foreach ($produits as $produit) {
    $produit['prixTTC'] = $produit['prix'] * 1.21; // ne modifie rien
}

// ✅ Utiliser array_map pour transformer
$produits = array_map(function(array $p): array {
    $p['prixTTC'] = $p['prix'] * 1.21;
    return $p;
}, $produits);

// ❌ Oublier array_values après array_filter
$filtre = array_filter($produits, fn($p) => $p['actif']);
echo $filtre[0]['nom']; // ❌ Peut ne pas exister si l'index 0 était inactif

// ✅ Réindexer après array_filter
$filtre = array_values(array_filter($produits, fn($p) => $p['actif']));
echo $filtre[0]['nom']; // ✅ Toujours l'index 0

// ❌ in_array sans strict sur des IDs
$ids = [1, 2, 3];
if (in_array("1", $ids)) { } // true alors que c'est un string !

// ✅ Toujours strict sur les IDs et valeurs sensibles
if (in_array(1, $ids, true)) { }
```

## 4. Ce que font les seniors 👨‍💻

Les seniors **chaînent** les opérations et évitent les variables intermédiaires inutiles :

```php
<?php
declare(strict_types=1);

/**
 * Retourne les produits actifs, enrichis, triés par prix
 *
 * @param array $produits Liste complète des produits
 * @return array          Produits traités et triés
 */
function preparerCatalogue(array $produits): array
{
    // Filtrer
    $actifs = array_values(array_filter(
        $produits,
        fn(array $p): bool => $p['actif']
    ));

    // Enrichir
    $enrichis = array_map(function(array $p): array {
        $p['disponibilite'] = $p['stock'] > 0 ? "En stock" : "Rupture";
        $p['prixTTC']       = round($p['prix'] * 1.21, 2);
        return $p;
    }, $actifs);

    // Trier
    usort($enrichis, fn(array $a, array $b): int => $a['prix'] <=> $b['prix']);

    return $enrichis;
}

/**
 * Calcule la valeur totale du stock
 *
 * @param array $produits Produits avec clés 'prix' et 'stock'
 * @return float          Valeur totale
 */
function calculerValeurStock(array $produits): float
{
    return array_reduce(
        $produits,
        fn(float $total, array $p): float => $total + ($p['prix'] * $p['stock']),
        0.0
    );
}
```

Tu travailles pour une **agence RH**. Tu reçois une liste de candidats à traiter.

Crée un fichier `candidats.php` avec ce tableau de départ :

```php
$candidats = [
    ["nom" => "Martin",  "age" => 17, "experience" => 0, "disponible" => true,  "salaire_souhaite" => 1800.0],
    ["nom" => "Bernard", "age" => 28, "experience" => 5, "disponible" => true,  "salaire_souhaite" => 3200.0],
    ["nom" => "Dupont",  "age" => 35, "experience" => 8, "disponible" => false, "salaire_souhaite" => 4100.0],
```

Avec des **fonctions** et les fonctions natives de tableau :

1. Une fonction `filtrerCandidatsValides(array $candidats): array` qui garde uniquement les candidats :
    - Disponibles
    - Majeurs (`age >= 18`)
    - Utilise `array_filter` + `array_values`
2. Une fonction `enrichirCandidats(array $candidats): array` qui ajoute à chaque candidat :
    - `profil` : via `match` sur l'expérience :
        - `0` → `"Junior"`
        - `1` à `3` → `"Débutant"`
        - `4` à `7` → `"Confirmé"`
        - `>= 8` → `"Senior"`
    - `salaire_estime` : `salaire_souhaite * 1.30` (coût employeur)
    - Utilise `array_map`
3. Une fonction `trierParExperience(array $candidats): array` qui trie par expérience **décroissante** avec `usort`
4. Une fonction `calculerMasseSalariale(array $candidats): float` qui calcule la somme des `salaire_estime` avec `array_reduce`
5. Dans le code principal :
    - Chaîne les fonctions dans le bon ordre
    - Affiche les candidats dans un tableau HTML
    - Affiche la masse salariale totale sous le tableau
    - Affiche le nombre de candidats par profil (Junior, Débutant, Confirmé, Senior)

```php
<?php
declare(strict_types=1);

/**
 * candidats.php
 * Gestion des candidats pour une agence RH
 */

// ============================================================
// DONNÉES
// ============================================================

$candidats = [
    ["nom" => "Martin",  "age" => 17, "experience" => 0,  "disponible" => true,  "salaire_souhaite" => 1800.0],
    ["nom" => "Bernard", "age" => 28, "experience" => 5,  "disponible" => true,  "salaire_souhaite" => 3200.0],
    ["nom" => "Dupont",  "age" => 35, "experience" => 8,  "disponible" => false, "salaire_souhaite" => 4100.0],
    ["nom" => "Leroy",   "age" => 22, "experience" => 1,  "disponible" => true,  "salaire_souhaite" => 2100.0],
    ["nom" => "Moreau",  "age" => 45, "experience" => 15, "disponible" => true,  "salaire_souhaite" => 5200.0],
    ["nom" => "Simon",   "age" => 19, "experience" => 0,  "disponible" => true,  "salaire_souhaite" => 1900.0],
];

// ============================================================

// ============================================================

/**
 * Filtre les candidats valides (disponibles et majeurs)
 *
 * @param array $candidats Liste complète des candidats
 * @return array           Candidats valides réindexés
 */
function filtrerCandidatsValides(array $candidats): array
{
    return array_values(array_filter(
        $candidats,
        fn(array $p): bool => $p['disponible'] === true && $p['age'] >= 18
    ));
}

/**
 * Enrichit chaque candidat avec son profil et son salaire estimé
 *
 * @param array $candidats Candidats à enrichir
 * @return array           Candidats enrichis
 */
function enrichirCandidats(array $candidats): array
{
    return array_map(function(array $p): array {
        $p['profil'] = match(true) {
            $p['experience'] === 0                          => 'Junior',
            $p['experience'] >= 1 && $p['experience'] <= 3 => 'Débutant',
            $p['experience'] >= 4 && $p['experience'] <= 7 => 'Confirmé',
            default                                         => 'Senior'
        };
        $p['salaire_estime'] = $p['salaire_souhaite'] * 1.30;
        return $p;
    }, $candidats);
}

/**
 * Trie les candidats par expérience décroissante
 *
 * @param array $candidats Candidats à trier
 * @return array           Candidats triés
 */
function trierParExperience(array $candidats): array
{
    usort($candidats, fn(array $a, array $b): int => $b['experience'] <=> $a['experience']);
    return $candidats;
}

/**
 * Calcule la masse salariale totale estimée
 *
 * @param array $candidats Candidats avec clé 'salaire_estime'
 * @return float           Masse salariale totale
 */
function calculerMasseSalariale(array $candidats): float
{
    return array_reduce(
        $candidats,
        fn(float $total, array $p): float => $total + $p['salaire_estime'],
        0.0
    );
}

/**
 * Compte les candidats par profil
 *
 * @param array $candidats Candidats avec clé 'profil'
 * @return array           Tableau associatif profil => nombre
 */
function compterParProfil(array $candidats): array
{
    $compteurs = [
        'Junior'   => 0,
        'Débutant' => 0,
        'Confirmé' => 0,
        'Senior'   => 0,
    ];

    foreach ($candidats as $candidat) {
        $compteurs[$candidat['profil']]++;
    }

    return $compteurs;
}

// ============================================================

$candidatsValides  = filtrerCandidatsValides($candidats);

$candidatsTries    = trierParExperience($candidatsEnrichis);
$masseSalariale    = calculerMasseSalariale($candidatsTries);
$parProfil         = compterParProfil($candidatsTries);
```
