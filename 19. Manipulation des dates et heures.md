# Étape 16 — Manipulation des dates et heures

## 1. La théorie complète

### Les fonctions de base

```php
<?php
// time() — timestamp Unix — nombre de secondes depuis le 01/01/1970
$timestamp = time();
echo $timestamp;   // 1742860800 (exemple)

// date() — formate un timestamp en chaîne lisible
// date(format, timestamp) — timestamp optionnel, défaut = maintenant
echo date('Y');           // "2026"
echo date('d/m/Y');       // "25/03/2026"
echo date('H:i:s');       // "14:30:00"
echo date('d/m/Y H:i');   // "25/03/2026 14:30"
```

### **Les formats de `date()`**

```php
// ── DATES ───────────────────────────────────────────────────
date('d')    // Jour avec zéro : "05", "25"
date('j')    // Jour sans zéro : "5", "25"
date('D')    // Jour abrégé EN : "Mon", "Tue"
date('l')    // Jour complet EN : "Monday", "Tuesday"
date('N')    // Numéro du jour : "1"=Lundi, "7"=Dimanche
date('m')    // Mois avec zéro : "01", "12"
date('n')    // Mois sans zéro : "1", "12"
date('M')    // Mois abrégé EN : "Jan", "Dec"
date('F')    // Mois complet EN : "January", "December"
date('Y')    // Année 4 chiffres : "2026"
date('y')    // Année 2 chiffres : "26"

// ── HEURES ──────────────────────────────────────────────────
date('H')    // Heure 24h avec zéro : "09", "23"
date('G')    // Heure 24h sans zéro : "9", "23"
date('i')    // Minutes avec zéro : "05", "59"
date('s')    // Secondes avec zéro : "00", "59"

// ── COMBINAISONS UTILES ─────────────────────────────────────
date('Y-m-d')           // "2026-03-25" — format ISO (base de données)
date('d/m/Y')           // "25/03/2026" — format français
date('d/m/Y H:i')       // "25/03/2026 14:30"
date('Y-m-d H:i:s')     // "2026-03-25 14:30:00" — datetime SQL
date('U')               // Timestamp Unix — identique à time()
```

### **`strtotime()` — convertir une chaîne en timestamp**

```php
// Convertit une date ou expression en timestamp
$ts = strtotime('2026-03-25');
echo date('d/m/Y', $ts);   // "25/03/2026"

// Expressions relatives — très puissant
echo date('d/m/Y', strtotime('+1 day'));       // Demain
echo date('d/m/Y', strtotime('+1 week'));      // Dans 1 semaine
echo date('d/m/Y', strtotime('+1 month'));     // Dans 1 mois
echo date('d/m/Y', strtotime('+1 year'));      // Dans 1 an
echo date('d/m/Y', strtotime('-30 days'));     // Il y a 30 jours
echo date('d/m/Y', strtotime('next Monday')); // Lundi prochain
echo date('d/m/Y', strtotime('last Friday')); // Vendredi dernier
echo date('d/m/Y', strtotime('first day of next month')); // 1er du mois prochain

// À partir d'une date existante
$dateInscription = '2026-01-15';
$expiration      = strtotime('+1 year', strtotime($dateInscription));
echo date('d/m/Y', $expiration);   // "15/01/2027"
```

### **`DateTime` — la classe moderne (recommandée)**

```php
// Créer un objet DateTime
$date = new DateTime();                    // Maintenant
$date = new DateTime('2026-03-25');        // Date spécifique
$date = new DateTime('2026-03-25 14:30');  // Date + heure

// Formater
echo $date->format('d/m/Y');         // "25/03/2026"
echo $date->format('Y-m-d H:i:s');   // "2026-03-25 14:30:00"

// Modifier
$date->modify('+1 month');
$date->modify('-2 days');
$date->modify('next Monday');

// Comparer deux dates avec diff()
$debut = new DateTime('2026-01-01');
$fin   = new DateTime('2026-03-25');
$diff  = $debut->diff($fin);

echo $diff->days;    // Nombre total de jours : 83
echo $diff->m;       // Mois : 2
echo $diff->d;       // Jours restants : 24
echo $diff->y;       // Années : 0

// Créer depuis un format spécifique
$date = DateTime::createFromFormat('d/m/Y', '25/03/2026');
$date = DateTime::createFromFormat('d-m-Y H:i', '25-03-2026 14:30');
echo $date->format('Y-m-d');   // "2026-03-25"
```

### **`DateTimeImmutable` — version immuable**

```php
// DateTimeImmutable — ne modifie PAS l'objet original
// Retourne un NOUVEL objet à chaque modification

$date    = new DateTimeImmutable('2026-03-25');
$demain  = $date->modify('+1 day');   // $date inchangé !

echo $date->format('d/m/Y');     // "25/03/2026" — inchangé
echo $demain->format('d/m/Y');   // "26/03/2026" — nouveau objet

// Préférer DateTimeImmutable — évite les bugs de mutation
```

### **Calculs de dates courants**

```php
<?php
declare(strict_types=1);

// ── Âge à partir d'une date de naissance ────────────────────
function calculerAge(string $dateNaissance): int
{
    $naissance = new DateTimeImmutable($dateNaissance);
    $aujourdhui = new DateTimeImmutable();
    return (int)$naissance->diff($aujourdhui)->y;
}

echo calculerAge('1992-05-15');   // 33

// ── Nombre de jours entre deux dates ────────────────────────
function joursEntre(string $dateDebut, string $dateFin): int
{
    $debut = new DateTimeImmutable($dateDebut);
    $fin   = new DateTimeImmutable($dateFin);
    return (int)$debut->diff($fin)->days;
}

echo joursEntre('2026-01-01', '2026-12-31');   // 364

// ── Vérifier si une date est passée ─────────────────────────
function estPassee(string $date): bool
{
    return new DateTimeImmutable($date) < new DateTimeImmutable();
}

// ── Vérifier si une date est dans le futur ──────────────────
function estFuture(string $date): bool
{
    return new DateTimeImmutable($date) > new DateTimeImmutable();
}

// ── Premier et dernier jour du mois ─────────────────────────
$premierJour = new DateTimeImmutable('first day of this month');
$dernierJour = new DateTimeImmutable('last day of this month');

echo $premierJour->format('d/m/Y');   // "01/03/2026"
echo $dernierJour->format('d/m/Y');   // "31/03/2026"
```

## 2. Exemple concret et utile

Un système de **gestion d'abonnements** :

```php
<?php
declare(strict_types=1);

date_default_timezone_set('Europe/Brussels');

/**
 * Calcule les informations d'un abonnement
 *
 * @param string $dateDebut    Date de début Y-m-d
 * @param int    $dureeMois    Durée en mois
 * @return array               Infos complètes de l'abonnement
 */
function calculerAbonnement(string $dateDebut, int $dureeMois): array
{
    $debut      = new DateTimeImmutable($dateDebut);
    $fin        = $debut->modify("+{$dureeMois} months");
    $maintenant = new DateTimeImmutable();

    $estActif       = $maintenant >= $debut && $maintenant <= $fin;
    $joursRestants  = $estActif ? (int)$maintenant->diff($fin)->days : 0;
    $joursEcoules   = (int)$debut->diff($maintenant)->days;
    $progression    = min(100, round(($joursEcoules / ($dureeMois * 30)) * 100));

    return [
        'date_debut'      => $debut->format('d/m/Y'),
        'date_fin'        => $fin->format('d/m/Y'),
        'est_actif'       => $estActif,
        'jours_restants'  => $joursRestants,
        'jours_ecoules'   => $joursEcoules,
        'progression'     => $progression,
        'statut'          => $estActif
            ? ($joursRestants <= 7 ? 'expire_bientot' : 'actif')
            : ($maintenant < $debut ? 'futur' : 'expire')
    ];
}

// Utilisation
$abonnement = calculerAbonnement('2026-01-01', 12);

echo "Début : "          . $abonnement['date_debut']     . "\n";
echo "Fin : "            . $abonnement['date_fin']       . "\n";
echo "Statut : "         . $abonnement['statut']         . "\n";
echo "Jours restants : " . $abonnement['jours_restants'] . "\n";
echo "Progression : "    . $abonnement['progression']    . "%\n";
```

## 3. Bonnes pratiques et erreurs à éviter

```php
// ❌ Stocker des dates en string non standard
$date = "25/03/2026";
// strtotime("25/03/2026") → false ! Format non reconnu

// ✅ Toujours stocker en format ISO
$date = "2026-03-25";   // Y-m-d — reconnu partout

// ❌ Utiliser date() sans configurer le fuseau
echo date('H:i');   // Heure du serveur — peut être UTC !

// ✅ Configurer le fuseau en config.php
date_default_timezone_set('Europe/Brussels');

// ❌ Comparer des dates comme des strings
if ("2026-03-25" > "2026-12-01") { }   // Fonctionne par hasard
// En format d/m/Y ça ne fonctionnerait plus

// ✅ Comparer avec DateTime
$d1 = new DateTimeImmutable('2026-03-25');
$d2 = new DateTimeImmutable('2026-12-01');
if ($d1 < $d2) { }   // ✅ Toujours correct

// ❌ DateTime — mutation silencieuse
$date = new DateTime('2026-03-25');
$date->modify('+1 month');   // Modifie $date directement — peut surprendre

// ✅ DateTimeImmutable — toujours préférer
$date    = new DateTimeImmutable('2026-03-25');
$moisPlus = $date->modify('+1 month');   // $date intact
```

## 4. Ce que font les seniors 👨‍💻

Les seniors centralisent les dates dans un helper et utilisent **toujours** `DateTimeImmutable` :

```php
<?php
// src/helpers/dates.php
declare(strict_types=1);

/**
 * Formate une date ISO en format français
 *
 * @param string $date    Date Y-m-d ou Y-m-d H:i:s
 * @param bool   $avecHeure Inclure l'heure
 * @return string          "25 mars 2026" ou "25 mars 2026 à 14h30"
 */
function formaterDateFr(string $date, bool $avecHeure = false): string
{
    $moisFr = [
        1  => 'janvier',  2  => 'février',   3  => 'mars',
        4  => 'avril',    5  => 'mai',        6  => 'juin',
        7  => 'juillet',  8  => 'août',       9  => 'septembre',
        10 => 'octobre',  11 => 'novembre',   12 => 'décembre'
    ];

    $dt    = new DateTimeImmutable($date);
    $mois  = $moisFr[(int)$dt->format('n')];
    $base  = $dt->format('j') . ' ' . $mois . ' ' . $dt->format('Y');

    if ($avecHeure) {
        return $base . ' à ' . $dt->format('H') . 'h' . $dt->format('i');
    }

    return $base;
}

/**
 * Retourne une date relative lisible
 * "il y a 5 minutes", "il y a 2 jours", etc.
 *
 * @param string $date Date Y-m-d H:i:s
 * @return string      Texte relatif
 */
function dateRelative(string $date): string
{
    $alors      = new DateTimeImmutable($date);
    $maintenant = new DateTimeImmutable();
    $diff       = $alors->diff($maintenant);

    if ($diff->y > 0) {
        return "il y a " . $diff->y . " an" . ($diff->y > 1 ? 's' : '');
    }
    if ($diff->m > 0) {
        return "il y a " . $diff->m . " mois";
    }
    if ($diff->d > 0) {
        return "il y a " . $diff->d . " jour" . ($diff->d > 1 ? 's' : '');
    }
    if ($diff->h > 0) {
        return "il y a " . $diff->h . " heure" . ($diff->h > 1 ? 's' : '');
    }
    if ($diff->i > 0) {
        return "il y a " . $diff->i . " minute" . ($diff->i > 1 ? 's' : '');
    }

    return "à l'instant";
}

/**
 * Calcule l'âge à partir d'une date de naissance
 *
 * @param string $dateNaissance Date Y-m-d
 * @return int                  Âge en années
 */
function calculerAge(string $dateNaissance): int
{
    return (int)(new DateTimeImmutable($dateNaissance))
        ->diff(new DateTimeImmutable())
        ->y;
}
```

### **`src/helpers/functions.php` — mise à jour complète**

```php
<?php
declare(strict_types=1);

/**
 * functions.php
 * Fonctions utilitaires — chaînes, dates, formatage
 */

// ============================================================
// CHAÎNES
// ============================================================

/**
 * Tronque un texte à une longueur donnée
 *
 * @param string $texte    Texte à tronquer
 * @param int    $longueur Longueur maximale en caractères
 * @return string          Texte tronqué ou original
 */
function tronquer(string $texte, int $longueur = 100): string
{
    if (strlen($texte) <= $longueur) {
        return $texte;
    }
    return substr($texte, 0, $longueur) . '...';
}

/**
 * Génère un slug URL-friendly à partir d'un titre
 *
 * @param string $titre Titre brut
 * @return string       Slug en minuscules sans accents
 */
function genererSlug(string $titre): string
{
    $slug = strtolower(trim($titre));
    $slug = str_replace(
        ['à','â','é','è','ê','ë','î','ï','ô','ù','û','ü','ç',' '],
        ['a','a','e','e','e','e','i','i','o','u','u','u','c','-'],
        $slug
    );
    // Supprimer les caractères non autorisés restants
    $slug = preg_replace('/[^a-z0-9-]/', '', $slug);
    // Supprimer les tirets multiples
    $slug = preg_replace('/-+/', '-', $slug);
    return trim($slug, '-');
}

// ============================================================
// DATES
// ============================================================

/**
 * Tableau de traduction des mois en français
 * Utilisé par plusieurs fonctions de formatage
 */
function getMoisFr(): array
{
    return [
        1  => 'janvier',   2  => 'février',   3  => 'mars',
        4  => 'avril',     5  => 'mai',        6  => 'juin',
        7  => 'juillet',   8  => 'août',       9  => 'septembre',
        10 => 'octobre',   11 => 'novembre',   12 => 'décembre'
    ];
}

/**
 * Formate une date ISO en format français lisible
 *
 * @param string $date      Date au format Y-m-d ou Y-m-d H:i:s
 * @param bool   $avecHeure Inclure l'heure dans le résultat
 * @return string           "25 mars 2026" ou "25 mars 2026 à 14h30"
 */
function formaterDateFr(string $date, bool $avecHeure = false): string
{
    $dt = new DateTimeImmutable($date);

    if ($dt === false) {
        return 'Date invalide';
    }

    $moisFr = getMoisFr();
    $mois   = $moisFr[(int)$dt->format('n')];
    $base   = $dt->format('j') . ' ' . $mois . ' ' . $dt->format('Y');

    if ($avecHeure) {
        return $base . ' à ' . $dt->format('H') . 'h' . $dt->format('i');
    }

    return $base;
}

/**
 * Alias de formaterDateFr pour compatibilité avec le code existant
 *
 * @param string $date Date au format Y-m-d
 * @return string      Date formatée en français
 */
function formaterDate(string $date): string
{
    return formaterDateFr($date);
}

/**
 * Retourne une date relative lisible en français
 * "à l'instant", "il y a 5 minutes", "il y a 2 jours", etc.
 *
 * @param string $date Date au format Y-m-d H:i:s ou Y-m-d
 * @return string      Texte relatif
 */
function dateRelative(string $date): string
{
    $alors      = new DateTimeImmutable($date);
    $maintenant = new DateTimeImmutable();
    $diff       = $alors->diff($maintenant);

    // On retourne dès qu'on trouve l'unité appropriée
    if ($diff->y > 0) {
        $n = $diff->y;
        return "il y a {$n} an" . ($n > 1 ? 's' : '');
    }

    if ($diff->m > 0) {
        $n = $diff->m;
        return "il y a {$n} mois";
    }

    if ($diff->d > 0) {
        $n = $diff->d;
        return "il y a {$n} jour" . ($n > 1 ? 's' : '');
    }

    if ($diff->h > 0) {
        $n = $diff->h;
        return "il y a {$n} heure" . ($n > 1 ? 's' : '');
    }

    if ($diff->i > 0) {
        $n = $diff->i;
        return "il y a {$n} minute" . ($n > 1 ? 's' : '');
    }

    return "à l'instant";
}

/**
 * Calcule l'âge en années à partir d'une date de naissance
 *
 * @param string $dateNaissance Date de naissance Y-m-d
 * @return int                  Âge en années révolues
 */
function calculerAge(string $dateNaissance): int
{
    return (int)(new DateTimeImmutable($dateNaissance))
        ->diff(new DateTimeImmutable())
        ->y;
}

/**
 * Calcule le nombre d'années depuis une date
 * Utilisé pour "Membre depuis X ans"
 *
 * @param string $date Date Y-m-d
 * @return int         Nombre d'années
 */
function anneesSince(string $date): int
{
    return (int)(new DateTimeImmutable($date))
        ->diff(new DateTimeImmutable())
        ->y;
}

/**
 * Calcule le prochain anniversaire à partir d'une date de naissance
 * Retourne la date du prochain anniv et le nombre de jours restants
 *
 * @param string $dateNaissance Date de naissance Y-m-d
 * @return array                ['date' => '...', 'dans_jours' => int]
 */
function prochainAnniversaire(string $dateNaissance): array
{
    $maintenant  = new DateTimeImmutable();
    $naissance   = new DateTimeImmutable($dateNaissance);

    // Anniversaire cette année
    $annivCetteAnnee = new DateTimeImmutable(
        $maintenant->format('Y') . '-' . $naissance->format('m-d')
    );

    // Si l'anniversaire est déjà passé cette année → prendre l'an prochain
    if ($annivCetteAnnee < $maintenant) {
        $annivCetteAnnee = $annivCetteAnnee->modify('+1 year');
    }

    $joursRestants = (int)$maintenant->diff($annivCetteAnnee)->days;

    return [
        'date'       => $annivCetteAnnee->format('d/m/Y'),
        'dans_jours' => $joursRestants,
        'est_auj'    => $joursRestants === 0
    ];
}

// ============================================================
// RECHERCHE (depuis crud.php — utilisée dans les pages)
// ============================================================

/**
 * Recherche des utilisateurs par ID
 * Utilisée dans profil.php
 *
 * @param int   $id    ID recherché
 * @param array $users Tableau des utilisateurs
 * @return array       L'utilisateur trouvé
 * @throws ValidationException Si ID invalide
 * @throws NotFoundException   Si non trouvé
 */
function trouverUtilisateur(int $id, array $users): array
{
    if ($id <= 0) {
        throw new ValidationException(
            "L'ID utilisateur doit être un entier positif. Reçu : {$id}",
            400
        );
    }

    foreach ($users as $user) {
        if ($user['id'] === $id) {
            return $user;
        }
    }

    throw new NotFoundException(
        "Aucun utilisateur trouvé avec l'ID {$id}",
        404
    );
}
```

```
┌─────────────────────────────────────────────────────────────┐
│  DATES EN PHP — HIÉRARCHIE                                  │
│                                                             │
│  time()           → timestamp Unix brut                     │
│  date()           → formater un timestamp                   │
│  strtotime()      → convertir string en timestamp           │
│                                                             │
│  DateTime         → objet mutable (attention aux bugs)      │
│  DateTimeImmutable→ objet immuable ✅ TOUJOURS PRÉFÉRER     │
│                                                             │
├─────────────────────────────────────────────────────────────┤
│  RÈGLES D'OR                                                │
│                                                             │
│  1. date_default_timezone_set() dans config.php             │
│  2. Stocker en base au format Y-m-d ou Y-m-d H:i:s         │
│  3. DateTimeImmutable plutôt que DateTime                   │
│  4. diff() pour comparer deux dates                         │
│  5. modify() pour ajouter/soustraire du temps               │
│                                                             │
├─────────────────────────────────────────────────────────────┤
│  FORMATS À RETENIR                                          │
│                                                             │
│  'Y-m-d'         → 2026-03-25  (BDD, ISO)                  │
│  'Y-m-d H:i:s'   → 2026-03-25 14:30:00  (datetime BDD)     │
│  'd/m/Y'         → 25/03/2026  (affichage FR)              │
│  'd/m/Y H:i'     → 25/03/2026 14:30  (affichage FR+heure)  │
└─────────────────────────────────────────────────────────────┘
```
