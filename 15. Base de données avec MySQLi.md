# Base de données avec MySQLi

## 1. La théorie complète

### C'est quoi MySQLi ?

MySQLi = **MySQL Improved** — l'extension PHP pour communiquer avec une base de données MySQL. Le "i" signifie "improved" par rapport à l'ancienne extension `mysql_*` qui est **supprimée depuis PHP 7**.

```php
PHP                    MySQLi                MySQL
 │                        │                    │
 │  mysqli_connect() ────►│                    │
 │                        │──── connexion ─────►│
 │                        │◄─── OK / Erreur ───│
 │  mysqli_query() ──────►│                    │
 │                        │──── SELECT ... ────►│
 │                        │◄─── résultats ─────│
 │◄─── données ──────────│                    │
```

### **Connexion à la base de données**

```php
<?php
declare(strict_types=1);

// mysqli_connect(hôte, utilisateur, mot_de_passe, base_de_données)
$connexion = mysqli_connect('localhost', 'root', '', 'ma_base');

// Vérifier si la connexion a réussi
if (!$connexion) {
    // mysqli_connect_error() retourne le message d'erreur
    die("Connexion échouée : " . mysqli_connect_error());
}

echo "Connexion réussie !";

// Définir l'encodage — TOUJOURS faire ça après la connexion
mysqli_set_charset($connexion, 'utf8mb4');
```

### Les opérations CRUD

CRUD = **C**reate, **R**ead, **U**pdate, **D**elete — les 4 opérations fondamentales.

### READ — Lire des données

```php
// mysqli_query() — exécute une requête
$resultat = mysqli_query($connexion, "SELECT * FROM utilisateurs");

// mysqli_fetch_assoc() — récupère une ligne sous forme de tableau associatif
while ($ligne = mysqli_fetch_assoc($resultat)) {
    echo $ligne['prenom'] . " " . $ligne['nom'] . "<br>";
}

// mysqli_fetch_all() — récupère TOUTES les lignes d'un coup
$resultat = mysqli_query($connexion, "SELECT * FROM utilisateurs");
$utilisateurs = mysqli_fetch_all($resultat, MYSQLI_ASSOC);

foreach ($utilisateurs as $user) {
    echo $user['prenom'] . "<br>";
}

// mysqli_num_rows() — nombre de résultats
$resultat = mysqli_query($connexion, "SELECT * FROM utilisateurs");
echo "Nombre d'utilisateurs : " . mysqli_num_rows($resultat);
```

### **CREATE — Insérer des données**

```php
// ❌ JAMAIS insérer des variables directement — injection SQL !
$nom = "Dupont";
mysqli_query($connexion, "INSERT INTO utilisateurs (nom) VALUES ('$nom')");
// Un attaquant peut envoyer : Dupont'); DROP TABLE utilisateurs; --

// ✅ Toujours utiliser les requêtes préparées
$stmt = mysqli_prepare($connexion, "INSERT INTO utilisateurs (prenom, nom, email) VALUES (?, ?, ?)");
//                                                                                          ↑  ↑  ↑
//                                                                          ? = placeholders sécurisés

// Lier les valeurs aux placeholders
// "sss" = trois strings (s=string, i=integer, d=double, b=blob)
mysqli_stmt_bind_param($stmt, "sss", $prenom, $nom, $email);

$prenom = "Jack";
$nom    = "Dupont";
$email  = "jack@test.com";

// Exécuter
mysqli_stmt_execute($stmt);

// Récupérer l'ID inséré
$idInsere = mysqli_insert_id($connexion);
echo "Nouvel utilisateur créé avec l'ID : " . $idInsere;

// Fermer le statement
mysqli_stmt_close($stmt);
```

### **UPDATE — Modifier des données**

```php
$stmt = mysqli_prepare(
    $connexion,
    "UPDATE utilisateurs SET prenom = ?, email = ? WHERE id = ?"
);

// "ssi" = string, string, integer
mysqli_stmt_bind_param($stmt, "ssi", $prenom, $email, $id);

$prenom = "Jacques";
$email  = "jacques@test.com";
$id     = 1;

mysqli_stmt_execute($stmt);

echo "{$lignesAffectees} ligne(s) modifiée(s)";

mysqli_stmt_close($stmt);
```

```php

// "i" = integer
```

### **Lire les résultats d'une requête préparée**

```php
$stmt = mysqli_prepare(
    $connexion,
    "SELECT id, prenom, nom, email FROM utilisateurs WHERE role = ?"
);

mysqli_stmt_bind_param($stmt, "s", $role);
$role = "admin";

mysqli_stmt_execute($stmt);

// Récupérer le résultat
$resultat = mysqli_stmt_get_result($stmt);

// Parcourir les résultats
while ($user = mysqli_fetch_assoc($resultat)) {
    echo $user['prenom'] . " " . $user['nom'] . "<br>";
}

mysqli_stmt_close($stmt);
```

### **Les types de bind_param**

```php
// "s" → string
// "i" → integer
// "d" → double (float)
// "b" → blob (données binaires)

// Exemples
mysqli_stmt_bind_param($stmt, "s",    $email);        // 1 string
mysqli_stmt_bind_param($stmt, "i",    $id);           // 1 integer
mysqli_stmt_bind_param($stmt, "si",   $email, $id);   // string + integer
mysqli_stmt_bind_param($stmt, "ssis", $prenom, $nom, $age, $email); // mixed
```

### **Fermer la connexion**

```php
// Fermer le statement après utilisation
mysqli_stmt_close($stmt);

// Fermer la connexion en fin de script
mysqli_close($connexion);
```

```php
/**
 * Retourne une connexion MySQLi
 * Pattern Singleton léger — une seule connexion par script
 *
 * @return \mysqli Connexion active
 * @throws DatabaseException Si la connexion échoue
 */
function getConnexion(): \mysqli
{
    static $connexion = null;

    // Si déjà connecté — retourner la connexion existante
    if ($connexion !== null) {
        return $connexion;
    }

    // Créer la connexion
    $connexion = mysqli_connect(DB_HOST, DB_USER, DB_PASS, DB_NAME);

    // Vérifier la connexion
    if (!$connexion) {
        throw new DatabaseException(
            "Connexion DB échouée : " . mysqli_connect_error(),
            500
        );
    }

    // Définir l'encodage UTF-8
    mysqli_set_charset($connexion, DB_CHARSET);

    return $connexion;
}
```

### **`src/helpers/crud.php` — fonctions CRUD réutilisables**

```php
<?php
declare(strict_types=1);

/**
 * crud.php
 * Fonctions CRUD génériques pour MySQLi
 */

/**
 * Récupère tous les enregistrements d'une table
 *
 * @param string $table Nom de la table
 * @return array        Tableau de résultats
 */
function fetchAll(string $table): array
{
    $connexion = getConnexion();
    // Nom de table sécurisé — pas de requête préparée possible sur les noms de tables
    // On valide donc le nom manuellement
    if (!preg_match('/^[a-zA-Z0-9_]+$/', $table)) {
        throw new ValidationException("Nom de table invalide : {$table}");
    }

    $resultat = mysqli_query($connexion, "SELECT * FROM `{$table}`");

    if ($resultat === false) {
        throw new DatabaseException("Erreur lors de la requête : " . mysqli_error($connexion));
    }

    return mysqli_fetch_all($resultat, MYSQLI_ASSOC);
}

/**
 * Récupère un enregistrement par son ID
 *
 * @param string $table Nom de la table
 * @param int    $id    ID de l'enregistrement
 * @return array        L'enregistrement trouvé
 * @throws NotFoundException Si introuvable
 */
function fetchById(string $table, int $id): array
{
    $connexion = getConnexion();

    $stmt = mysqli_prepare($connexion, "SELECT * FROM `{$table}` WHERE id = ?");
    mysqli_stmt_bind_param($stmt, "i", $id);
    mysqli_stmt_execute($stmt);

    $resultat    = mysqli_stmt_get_result($stmt);
    $enregistrement = mysqli_fetch_assoc($resultat);

    mysqli_stmt_close($stmt);

    if (!$enregistrement) {
        throw new NotFoundException("Enregistrement ID {$id} introuvable dans {$table}", 404);
    }

    return $enregistrement;
}

/**
 * Insère un enregistrement dans une table
 *
 * @param string $table   Nom de la table
 * @param array  $donnees Données à insérer ['colonne' => 'valeur']
 * @return int            ID de l'enregistrement inséré
 */
function insert(string $table, array $donnees): int
{
    $connexion = getConnexion();

    $colonnes    = implode(', ',  array_keys($donnees));
    $placeholders = implode(', ', array_fill(0, count($donnees), '?'));

    $sql  = "INSERT INTO `{$table}` ({$colonnes}) VALUES ({$placeholders})";
    $stmt = mysqli_prepare($connexion, $sql);

    // Construire les types dynamiquement
    $types  = buildTypes(array_values($donnees));
    $valeurs = array_values($donnees);

    mysqli_stmt_bind_param($stmt, $types, ...$valeurs);
    mysqli_stmt_execute($stmt);

    $idInsere = mysqli_insert_id($connexion);

    mysqli_stmt_close($stmt);

    return $idInsere;
}

/**
 * Met à jour un enregistrement par son ID
 *
 * @param string $table   Nom de la table
 * @param int    $id      ID de l'enregistrement
 * @param array  $donnees Données à mettre à jour
 * @return int            Nombre de lignes affectées
 */
function update(string $table, int $id, array $donnees): int
{
    $connexion = getConnexion();

    $set = implode(', ', array_map(
        fn(string $col): string => "`{$col}` = ?",
        array_keys($donnees)
    ));

    $sql  = "UPDATE `{$table}` SET {$set} WHERE id = ?";
    $stmt = mysqli_prepare($connexion, $sql);

    $valeurs  = array_values($donnees);
    $valeurs[] = $id;   // Ajouter l'ID à la fin
    $types    = buildTypes($valeurs);

    mysqli_stmt_bind_param($stmt, $types, ...$valeurs);
    mysqli_stmt_execute($stmt);

    $lignesAffectees = mysqli_affected_rows($connexion);
    mysqli_stmt_close($stmt);

    return $lignesAffectees;
}

/**
 * Supprime un enregistrement par son ID
 *
 * @param string $table Nom de la table
 * @param int    $id    ID de l'enregistrement
 * @return int          Nombre de lignes supprimées
 */
function delete(string $table, int $id): int
{
    $connexion = getConnexion();

    $stmt = mysqli_prepare($connexion, "DELETE FROM `{$table}` WHERE id = ?");
    mysqli_stmt_bind_param($stmt, "i", $id);
    mysqli_stmt_execute($stmt);

    $lignesAffectees = mysqli_affected_rows($connexion);
    mysqli_stmt_close($stmt);

    return $lignesAffectees;
}

/**
 * Construit la chaîne de types pour bind_param
 * s = string, i = integer, d = double
 *
 * @param array $valeurs Tableau de valeurs
 * @return string        Chaîne de types ("ssi", "iss", etc.)
 */
function buildTypes(array $valeurs): string
{
    return implode('', array_map(function($valeur): string {
        if (is_int($valeur))   return 'i';
        if (is_float($valeur)) return 'd';
        return 's';
    }, $valeurs));
}
```

## **4. Bonnes pratiques et erreurs à éviter**

```php
// ❌ Concaténer des variables dans les requêtes SQL
$sql = "SELECT * FROM users WHERE email = '$email'";
// Injection SQL possible !

// ✅ Toujours les requêtes préparées
$stmt = mysqli_prepare($connexion, "SELECT * FROM users WHERE email = ?");
mysqli_stmt_bind_param($stmt, "s", $email);

// ❌ Ne pas vérifier le retour de mysqli_query
mysqli_query($connexion, "UPDATE ...");
// Si ça échoue — aucune idée !

// ✅ Toujours vérifier
$resultat = mysqli_query($connexion, "UPDATE ...");
if ($resultat === false) {
    throw new DatabaseException(mysqli_error($connexion));
}

// ❌ Ouvrir plusieurs connexions
function getUsers() {
    $co = mysqli_connect(...);   // Nouvelle connexion à chaque appel !
}

// ✅ Une seule connexion — pattern singleton
$connexion = getConnexion();   // Réutilise la connexion existante
```

```php
if ($_SERVER['REQUEST_METHOD'] === 'POST') {

    $action = $_POST['action'] ?? '';

    if ($action === 'supprimer') {

            $erreur = htmlspecialchars($e->getMessage(), ENT_QUOTES, 'UTF-8');
        } catch (DatabaseException $e) {
            loggerErreur($e);
            $erreur = "Erreur base de données lors de la suppression.";
        }

    // ── CRÉATION ────────────────────────────────────────────
    if ($action === 'creer') {

        $titre   = trim($_POST['titre']   ?? '');

        $auteur  = trim($_POST['auteur']  ?? '');
        $statut  = $_POST['statut']       ?? 'brouillon';

        $erreurs = [];

        // Validation

            $erreurs[] = "Le titre est obligatoire";

            $erreurs[] = "Le titre doit faire au moins 3 caractères";

            $erreurs[] = "Le contenu est obligatoire";

            $erreurs[] = "L'auteur est obligatoire";

        if (!in_array($statut, ['publie', 'brouillon'], true)) {
            $statut = 'brouillon';
        }

        if (empty($erreurs)) {
            try {
                $id = insert('articles', [
                    'titre'   => $titre,
                    'contenu' => $contenu,
                    'auteur'  => $auteur,
                    'statut'  => $statut,
                ]);

                header('Location: ' . APP_URL . '?page=articles&succes=cree&id=' . $id);
                exit;

            } catch (DatabaseException $e) {
                loggerErreur($e);
                $erreur = "Erreur base de données lors de la création.";
            }
        }
    }
}
```
