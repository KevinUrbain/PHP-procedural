# Étape 11 — Gestion des erreurs

## 1. La théorie complète

### Les niveaux d'erreurs PHP

PHP distingue plusieurs types d'erreurs — du plus grave au moins grave :

```php
<?php
// E_ERROR — Erreur fatale — stoppe l'exécution
// Exemple : appeler une fonction qui n'existe pas
maFonctionInexistante();

// E_WARNING — Avertissement — continue l'exécution
// Exemple : inclure un fichier manquant avec include
include 'fichier_inexistant.php';

// E_NOTICE — Notice — information, continue l'exécution
// Exemple : utiliser une variable non définie
echo $variableIndefinie;

// E_DEPRECATED — Fonctionnalité obsolète
// Exemple : utiliser une fonction dépréciée
```

```php
// Configurer quelles erreurs afficher
error_reporting(E_ALL);              // Toutes les erreurs
error_reporting(E_ALL & ~E_NOTICE);  // Tout sauf les notices
error_reporting(0);                  // Aucune erreur affichée

// Afficher ou masquer les erreurs
ini_set('display_errors', '1');   // Afficher (développement)
ini_set('display_errors', '0');   // Masquer (production)

// Logger les erreurs dans un fichier
ini_set('log_errors', '1');
ini_set('error_log', __DIR__ . '/logs/errors.log');
```

### `die()` et `exit()` — arrêter l'exécution

```php
// exit() et die() sont identiques
// Elles stoppent l'exécution immédiatement

$fichier = fopen('data.txt', 'r');

if ($fichier === false) {
    die("Impossible d'ouvrir le fichier");
    // ou
    exit("Impossible d'ouvrir le fichier");
}

// exit() avec un code numérique — pour les scripts CLI
exit(0);    // Succès
exit(1);    // Erreur générique
```

### `try / catch / finally` — gestion structurée

En PHP procédural, `try/catch` s'utilise principalement avec les **exceptions** :

```php
// Lancer une exception
function diviser(float $a, float $b): float
{
    if ($b === 0.0) {
        throw new Exception("Division par zéro impossible");
    }
    return $a / $b;
}

// Attraper une exception
try {
    $resultat = diviser(10, 0);
    echo $resultat;
} catch (Exception $e) {
    echo "Erreur : " . $e->getMessage();
    // "Erreur : Division par zéro impossible"
} finally {
    // S'exécute TOUJOURS — avec ou sans exception
    echo "Calcul terminé";
}
```

### **Les méthodes de l'objet Exception**

```php
try {
    throw new Exception("Quelque chose a mal tourné", 404);
} catch (Exception $e) {
    echo $e->getMessage();    // "Quelque chose a mal tourné"
    echo $e->getCode();       // 404
    echo $e->getFile();       // Chemin du fichier où l'erreur s'est produite
    echo $e->getLine();       // Numéro de ligne
    echo $e->getTraceAsString(); // Stack trace complète
}
```

### **Attraper plusieurs types d'exceptions**

```php
// On peut créer ses propres exceptions
class ValidationException extends Exception {}
class DatabaseException extends Exception {}
class NotFoundException extends Exception {}

function trouverUtilisateur(int $id): array
{
    if ($id <= 0) {
        throw new ValidationException("L'ID doit être positif", 400);
    }

    // Simule une DB
    $users = [1 => ["nom" => "Jack"], 2 => ["nom" => "Alice"]];

    if (!isset($users[$id])) {
        throw new NotFoundException("Utilisateur {$id} introuvable", 404);
    }

    return $users[$id];
}

// Attraper chaque type différemment
try {
    $user = trouverUtilisateur(-1);
} catch (ValidationException $e) {
    echo "Erreur de validation : " . $e->getMessage();
} catch (NotFoundException $e) {
    echo "Introuvable : " . $e->getMessage();
} catch (Exception $e) {
    // Filet de sécurité — attrape tout le reste
    echo "Erreur inattendue : " . $e->getMessage();
} finally {
    echo "Recherche terminée";
}
```

### **Gestionnaire d'erreurs personnalisé**

```php
<?php
// Intercepter les erreurs PHP avec une fonction personnalisée
function gestionnaireErreurs(
    int    $niveau,
    string $message,
    string $fichier,
    int    $ligne
): bool {

    $niveauxTexte = [
        E_ERROR   => 'ERREUR FATALE',
        E_WARNING => 'AVERTISSEMENT',
        E_NOTICE  => 'NOTICE',
    ];

    $label = $niveauxTexte[$niveau] ?? 'ERREUR INCONNUE';

    // Logger dans un fichier
    $logMessage = sprintf(
        "[%s] %s : %s dans %s ligne %d\n",
        date('Y-m-d H:i:s'),
        $label,
        $message,
        $fichier,
        $ligne
    );

    error_log($logMessage, 3, __DIR__ . '/logs/errors.log');

    // En développement — afficher
    if (ENV === 'development') {
        echo "<p style='color:red'><strong>{$label}</strong> : {$message}</p>";
    }

    // Retourner true = PHP ne gère plus l'erreur lui-même
    return true;
}

// Enregistrer le gestionnaire
set_error_handler('gestionnaireErreurs');
```

```php
    }
}

set_exception_handler('gestionnaireExceptions');
```

## 2. Exemple concret et utile

Un système de **gestion d'erreurs complet** pour un projet professionnel :

```php
<?php
declare(strict_types=1);

/**
 * src/helpers/errors.php
 * Gestionnaire d'erreurs centralisé
 */

require_once __DIR__ . '/../config/config.php';

// ============================================================
// EXCEPTIONS PERSONNALISÉES
// ============================================================

class ValidationException extends Exception {}
class NotFoundException    extends Exception {}
class DatabaseException    extends Exception {}

// ============================================================
// GESTIONNAIRES
// ============================================================

/**
 * Gestionnaire d'erreurs PHP
 */
function gestionnaireErreurs(
    int    $niveau,
    string $message,
    string $fichier,
    int    $ligne
): bool {

    // Ignorer les erreurs supprimées avec @
    if (error_reporting() === 0) {
        return false;
    }

    throw new \ErrorException($message, 0, $niveau, $fichier, $ligne);
}

/**
 * Gestionnaire d'exceptions non attrapées
 */
function gestionnaireExceptions(Throwable $e): void
{
    loggerErreur($e);

    if (ENV === 'development') {
        afficherErreurDev($e);
    } else {
        afficherErreurProd();
    }
}

/**
 * Logger une erreur dans le fichier de log
 */
function loggerErreur(Throwable $e): void
{
    $dossierLogs = __DIR__ . '/../../logs';

    if (!is_dir($dossierLogs)) {
        mkdir($dossierLogs, 0755, true);
    }

    $message = sprintf(
        "[%s] %s : %s dans %s ligne %d\n",
        date('Y-m-d H:i:s'),
        get_class($e),
        $e->getMessage(),
        $e->getFile(),
        $e->getLine()
    );

    error_log($message, 3, $dossierLogs . '/errors.log');
}

/**
 * Affichage détaillé en développement
 */
function afficherErreurDev(Throwable $e): void
{
    http_response_code(500);
    echo "<style>body{font-family:monospace;padding:20px;background:#1e1e1e;color:#d4d4d4}</style>";
    echo "<h2 style='color:#f44747'>" . get_class($e) . "</h2>";
    echo "<p>" . htmlspecialchars($e->getMessage(), ENT_QUOTES, 'UTF-8') . "</p>";
    echo "<p style='color:#888'>Dans " . $e->getFile() . " ligne " . $e->getLine() . "</p>";
    echo "<pre style='color:#9cdcfe'>" . htmlspecialchars($e->getTraceAsString(), ENT_QUOTES, 'UTF-8') . "</pre>";
}

/**
 * Affichage générique en production
 */
function afficherErreurProd(): void
{
    http_response_code(500);
    echo "<h1>Une erreur est survenue. Veuillez réessayer plus tard.</h1>";
}

// Enregistrement des gestionnaires
set_error_handler('gestionnaireErreurs');
set_exception_handler('gestionnaireExceptions');
register_shutdown_function(function(): void {
    $erreur = error_get_last();
    if ($erreur && $erreur['type'] === E_ERROR) {
        afficherErreurProd();
    }
});
```

### **3. Bonnes pratiques et erreurs à éviter**

```php
// ❌ Attraper Exception et ne rien faire — bug silencieux
try {
    $resultat = operationRisquee();
} catch (Exception $e) {
    // rien — l'erreur est ignorée silencieusement !
}

// ✅ Toujours au minimum logger l'erreur
try {
    $resultat = operationRisquee();
} catch (Exception $e) {
    loggerErreur($e);
    // Puis gérer proprement
}

// ❌ Afficher les détails d'erreur en production
} catch (Exception $e) {
    echo $e->getMessage();   // Peut révéler la structure du code !
}

// ✅ Message générique en production, détails uniquement en dev
} catch (Exception $e) {
    loggerErreur($e);
    if (ENV === 'development') {
        echo $e->getMessage();
    } else {
        echo "Une erreur est survenue";
    }
}

// ❌ Utiliser die() partout sans logger
if (!$connexion) {
    die("Erreur DB");   // L'erreur est perdue
}

// ✅ Logger avant de mourir
if (!$connexion) {
    error_log("Connexion DB échouée : " . mysqli_connect_error());
    http_response_code(500);
    die("Service temporairement indisponible");
}
```

### **`src/helpers/errors.php`**

```php
<?php
declare(strict_types=1);

/**
 * errors.php
 * Gestionnaire d'erreurs centralisé
 */

// ============================================================
// EXCEPTIONS PERSONNALISÉES
// ============================================================

/**
 * Lancée quand les données reçues sont invalides
 * Exemple : ID négatif, email mal formaté
 */
class ValidationException extends Exception {}

/**
 * Lancée quand une ressource est introuvable
 * Exemple : utilisateur ID 99 n'existe pas
 */
class NotFoundException extends Exception {}

/**
 * Lancée quand l'accès est refusé
 * Exemple : page admin accédée sans être connecté
 */
class AuthException extends Exception {}

// ============================================================
// FONCTIONS DE GESTION
// ============================================================

/**
 * Ecrit l'erreur dans le fichier de log
 * Appelée dans chaque catch pour ne jamais perdre une erreur
 *
 * @param Throwable $e L'exception ou erreur attrapée
 */
function loggerErreur(Throwable $e): void
{
    // Créer le dossier logs s'il n'existe pas
    $dossierLogs = __DIR__ . '/../../logs';

    if (!is_dir($dossierLogs)) {
        mkdir($dossierLogs, 0755, true);
    }

    // Formater le message de log
    $message = sprintf(
        "[%s] %s (code %d) : %s dans %s ligne %d\n",
        date('Y-m-d H:i:s'),    // Horodatage
        get_class($e),           // Type d'exception
        $e->getCode(),           // Code erreur
        $e->getMessage(),        // Message
        $e->getFile(),           // Fichier
        $e->getLine()            // Ligne
    );

    // Ecrire dans le fichier — mode append (3)
    error_log($message, 3, $dossierLogs . '/errors.log');
}

/**
 * Affiche l'erreur selon l'environnement
 * En dev  : détails complets
 * En prod : message générique
 *
 * @param Throwable $e L'exception attrapée
 */
function afficherErreur(Throwable $e): void
{
    if (ENV === 'development') {
        // Détails complets pour le développeur
        echo "<div style='font-family:monospace;background:#1e1e1e;color:#d4d4d4;padding:20px;margin:10px'>";
        echo "<h2 style='color:#f44747'>" . get_class($e) . "</h2>";
        echo "<p><strong>Message :</strong> " . htmlspecialchars($e->getMessage(), ENT_QUOTES, 'UTF-8') . "</p>";
        echo "<p><strong>Fichier :</strong> " . htmlspecialchars($e->getFile(), ENT_QUOTES, 'UTF-8') . "</p>";
        echo "<p><strong>Ligne :</strong> " . $e->getLine() . "</p>";
        echo "<pre style='color:#9cdcfe'>" . htmlspecialchars($e->getTraceAsString(), ENT_QUOTES, 'UTF-8') . "</pre>";
        echo "</div>";
    } else {
        // Message générique en production — ne révèle rien
        echo "<p>Une erreur est survenue. Veuillez réessayer plus tard.</p>";
    }
}

/**
 * Gestionnaire d'exceptions non attrapées
 * Appelé automatiquement par PHP quand une exception
 * n'est attrapée dans aucun try/catch
 *
 * @param Throwable $e L'exception non attrapée
 */
function gestionnaireExceptions(Throwable $e): void
{
    http_response_code(500);
    loggerErreur($e);
    afficherErreur($e);
}

/**
 * Gestionnaire d'erreurs PHP classiques
 * Convertit les erreurs PHP en exceptions pour
 * les traiter de manière uniforme
 */
function gestionnaireErreurs(
    int    $niveau,
    string $message,
    string $fichier,
    int    $ligne
): bool {
    // Ignorer les erreurs supprimées avec @
    if (error_reporting() === 0) {
        return false;
    }

    // Convertir l'erreur PHP en exception
    throw new \ErrorException($message, 0, $niveau, $fichier, $ligne);
}

// ============================================================
// ENREGISTREMENT DES GESTIONNAIRES
// ============================================================

// Intercepte les erreurs PHP classiques
set_error_handler('gestionnaireErreurs');

// Intercepte les exceptions non attrapées
set_exception_handler('gestionnaireExceptions');

// S'exécute à la fin du script — même sur erreur fatale
register_shutdown_function(function(): void {
    $erreur = error_get_last();
    if ($erreur !== null && $erreur['type'] === E_ERROR) {
        loggerErreur(new \ErrorException(
            $erreur['message'],
            0,
            $erreur['type'],
            $erreur['file'],
            $erreur['line']
        ));
        if (ENV !== 'development') {
            echo "<p>Erreur critique. Veuillez réessayer.</p>";
        }
    }
});
```

### **`src/helpers/functions.php` — ajout de `trouverUtilisateur()`**

```php
<?php
declare(strict_types=1);

// ... tes fonctions existantes (formaterDate, tronquer, genererSlug) ...

/**
 * Recherche un utilisateur par son ID dans un tableau
 * Lance une exception si l'ID est invalide ou si l'utilisateur est introuvable
 *
 * @param int   $id    ID recherché
 * @param array $users Tableau des utilisateurs
 * @return array       L'utilisateur trouvé
 *
 * @throws ValidationException Si l'ID est invalide (<= 0)
 * @throws NotFoundException   Si aucun utilisateur ne correspond
 */
function trouverUtilisateur(int $id, array $users): array
{
    // Validation — l'ID doit être positif
    if ($id <= 0) {
        throw new ValidationException(
            "L'ID utilisateur doit être un entier positif. Reçu : {$id}",
            400
        );
    }

    // Recherche dans le tableau
    foreach ($users as $user) {
        if ($user['id'] === $id) {
            return $user;   // Trouvé — on retourne immédiatement
        }
    }

    // Pas trouvé — on lance l'exception
    throw new NotFoundException(
        "Aucun utilisateur trouvé avec l'ID {$id}",
        404
    );
}
```

```php
require_once __DIR__ . '/../src/helpers/errors.php';      // ← ajouté
require_once __DIR__ . '/../src/helpers/functions.php';

session_start();

// Nettoyage du paramètre page
$page   = preg_replace('/[^a-z0-9_-]/', '', $_GET['page'] ?? 'accueil');
$chemin = __DIR__ . '/../pages/' . $page . '.php';

// Pages autorisées — liste blanche
$pagesAutorisees = ['accueil', 'equipe', 'profil', 'contact', 'login'];  // ← profil ajouté

if (!in_array($page, $pagesAutorisees, true) || !file_exists($chemin)) {
    $chemin = __DIR__ . '/../pages/404.php';
    http_response_code(404);
}

// Rendu
require_once __DIR__ . '/../src/templates/header.php';
require_once $chemin;
require_once __DIR__ . '/../src/templates/footer.php';
```
