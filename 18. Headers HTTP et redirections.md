# Headers HTTP et redirections

# Étape 15 — Headers HTTP et redirections

## 1. La théorie complète

### C'est quoi un header HTTP ?

Quand un navigateur communique avec un serveur, il échange des **headers** — des métadonnées qui précèdent le contenu de la page.

```
Navigateur → Serveur :
GET /index.php HTTP/1.1
Host: monsite.com
Accept: text/html
Cookie: PHPSESSID=abc123

Serveur → Navigateur :
HTTP/1.1 200 OK
Content-Type: text/html; charset=UTF-8
Location: /dashboard.php    ← header de redirection
Set-Cookie: theme=dark
```

### **`header()` en PHP**

```php
// header() envoie un header HTTP
// DOIT être appelé AVANT tout output (echo, HTML, espaces...)

<?php
header('Location: https://google.com');
exit;   // ← TOUJOURS après un header Location
?>

// ❌ Trop tard — output déjà envoyé
echo "Bonjour";
header('Location: page.php');   // Warning : headers already sent
```

### **Les codes de statut HTTP**

```php
// Les plus importants à connaître

// 2xx — Succès
http_response_code(200);   // OK — par défaut
http_response_code(201);   // Created — ressource créée

// 3xx — Redirections
http_response_code(301);   // Moved Permanently — SEO
http_response_code(302);   // Found — redirection temporaire (défaut)
http_response_code(303);   // See Other — après un POST (pattern PRG)

// 4xx — Erreurs client
http_response_code(400);   // Bad Request — données invalides
http_response_code(401);   // Unauthorized — non authentifié
http_response_code(403);   // Forbidden — accès refusé
http_response_code(404);   // Not Found — page introuvable
http_response_code(422);   // Unprocessable Entity — validation échouée

// 5xx — Erreurs serveur
http_response_code(500);   // Internal Server Error
http_response_code(503);   // Service Unavailable — maintenance
```

### **Redirections**

```php
// Redirection simple
header('Location: /login.php');
exit;

// Redirection avec code explicite
header('Location: /dashboard.php', true, 303);
//                                  ↑     ↑
//                         remplacer    code HTTP
//                         le header

// Redirection permanente — pour le SEO
header('Location: https://nouveau-domaine.com', true, 301);
exit;

// Redirection avec message
header('Location: /articles.php?succes=cree');
exit;

// Redirection vers page précédente
$retour = $_SERVER['HTTP_REFERER'] ?? '/accueil.php';
header('Location: ' . $retour);
exit;
```

### **Content-Type — type de contenu**

```php
// HTML — défaut
header('Content-Type: text/html; charset=UTF-8');

// JSON — pour les APIs
header('Content-Type: application/json; charset=UTF-8');
echo json_encode(['succes' => true, 'message' => 'OK']);

// CSV — téléchargement
header('Content-Type: text/csv; charset=UTF-8');
header('Content-Disposition: attachment; filename="export.csv"');

// PDF
header('Content-Type: application/pdf');
header('Content-Disposition: inline; filename="document.pdf"');

// Forcer le téléchargement
header('Content-Type: application/octet-stream');
header('Content-Disposition: attachment; filename="fichier.zip"');
header('Content-Length: ' . filesize('fichier.zip'));
```

### **Cache HTTP**

```php
// Désactiver le cache — pages dynamiques sensibles
header('Cache-Control: no-store, no-cache, must-revalidate');
header('Pragma: no-cache');
header('Expires: 0');

// Autoriser le cache — ressources statiques
header('Cache-Control: public, max-age=86400');  // 1 jour
header('Cache-Control: public, max-age=2592000'); // 30 jours

// Cache conditionnel — vérifier si modifié
header('Last-Modified: ' . gmdate('D, d M Y H:i:s') . ' GMT');
header('ETag: "' . md5_file('style.css') . '"');
```

### **Fonction `headers_sent()`**

```php
// Vérifier si les headers ont déjà été envoyés
if (headers_sent($fichier, $ligne)) {
    echo "Headers envoyés depuis {$fichier} ligne {$ligne}";
} else {
    header('Location: /login.php');
    exit;
}
```

## 2. Exemple concret — mini API JSON

Un cas très fréquent en entreprise — répondre en JSON pour une API :

```php
<?php
declare(strict_types=1);

/**
 * api/articles.php
 * Mini API JSON pour les articles
 */

require_once __DIR__ . '/../../src/config/config.php';
require_once __DIR__ . '/../../src/helpers/errors.php';
require_once __DIR__ . '/../../src/config/database.php';
require_once __DIR__ . '/../../src/helpers/crud.php';

// Toujours répondre en JSON pour une API
header('Content-Type: application/json; charset=UTF-8');

// Pas de cache sur les données dynamiques
header('Cache-Control: no-store');

/**
 * Envoie une réponse JSON et arrête le script
 *
 * @param mixed $donnees    Données à encoder
 * @param int   $codeHTTP   Code HTTP de la réponse
 */
function repondreJSON(mixed $donnees, int $codeHTTP = 200): void
{
    http_response_code($codeHTTP);
    echo json_encode($donnees, JSON_UNESCAPED_UNICODE | JSON_PRETTY_PRINT);
    exit;
}

// Récupérer la méthode HTTP
$methode = $_SERVER['REQUEST_METHOD'];
$id      = (int)($_GET['id'] ?? 0);

try {

    // GET /api/articles.php — liste tous les articles
    if ($methode === 'GET' && $id === 0) {
        $result = fetchAll('articles', 'created_at');
        repondreJSON([
            'succes'   => true,
            'total'    => count($result['data']),
            'articles' => $result['data']
        ]);
    }

    // GET /api/articles.php?id=1 — récupère un article
    if ($methode === 'GET' && $id > 0) {
        $article = fetchById('articles', $id);
        repondreJSON(['succes' => true, 'article' => $article]);
    }

    // POST /api/articles.php — crée un article
    if ($methode === 'POST') {
        $body    = json_decode(file_get_contents('php://input'), true);
        $titre   = trim($body['titre']   ?? '');
        $contenu = trim($body['contenu'] ?? '');
        $auteur  = trim($body['auteur']  ?? '');

        if (empty($titre) || empty($contenu) || empty($auteur)) {
            repondreJSON(['succes' => false, 'erreur' => 'Champs manquants'], 422);
        }

        $newId = insert('articles', [
            'titre'   => $titre,
            'contenu' => $contenu,
            'auteur'  => $auteur,
            'statut'  => 'brouillon'
        ]);

        repondreJSON(['succes' => true, 'id' => $newId], 201);
    }

    repondreJSON(['succes' => false, 'erreur' => 'Méthode non supportée'], 405);

} catch (NotFoundException $e) {
    repondreJSON(['succes' => false, 'erreur' => $e->getMessage()], 404);
} catch (Exception $e) {
    loggerErreur($e);
    repondreJSON(['succes' => false, 'erreur' => 'Erreur serveur'], 500);
}
```

### **3. Bonnes pratiques et erreurs à éviter**

```php
// ❌ Oublier exit après header Location
header('Location: /login.php');
// Le script continue de s'exécuter !
// Peut causer des fuites de données ou des bugs

// ✅ Toujours exit après une redirection
header('Location: /login.php');
exit;

// ❌ Output avant header
echo "Début de page";
header('Location: /login.php');   // Warning !

// ✅ Aucun output avant header
<?php
header('Location: /login.php');
exit;
?>

// ❌ Redirection sans code HTTP explicite après POST
header('Location: /articles.php');
// Le navigateur peut refaire le POST si l'utilisateur navigue

// ✅ Code 303 après POST — force un GET
header('Location: /articles.php', true, 303);
exit;

// ❌ Exposer des infos dans les headers
header('X-Powered-By: PHP/8.2');   // Révèle la version PHP !
header('Server: Apache/2.4');       // Révèle le serveur !

// ✅ Masquer ces infos dans php.ini
// expose_php = Off
```

## 4. Ce que font les seniors 👨‍💻

Les seniors centralisent les redirections dans des helpers :

```php
<?php
// src/helpers/http.php
declare(strict_types=1);

/**
 * Redirige vers une URL avec un code HTTP
 *
 * @param string $url      URL de destination
 * @param int    $code     Code HTTP (303 par défaut après POST)
 */
function rediriger(string $url, int $code = 303): void
{
    if (headers_sent()) {
        // Fallback JavaScript si les headers sont déjà envoyés
        echo "<script>window.location.href='" . addslashes($url) . "'</script>";
        exit;
    }
    header('Location: ' . $url, true, $code);
    exit;
}

/**
 * Redirige avec un message flash en session
 * Le message sera affiché sur la page suivante
 *
 * @param string $url     URL de destination
 * @param string $message Message à afficher
 * @param string $type    Type : 'succes', 'erreur', 'info'
 */
function redigirerAvecMessage(string $url, string $message, string $type = 'succes'): void
{
    $_SESSION['flash'] = [
        'message' => $message,
        'type'    => $type
    ];
    rediriger($url);
}

/**
 * Récupère et supprime le message flash de la session
 *
 * @return array|null ['message' => '...', 'type' => '...'] ou null
 */
function getMessageFlash(): ?array
{
    if (!isset($_SESSION['flash'])) {
        return null;
    }
    $flash = $_SESSION['flash'];
    unset($_SESSION['flash']);  // Supprimer après lecture — one-shot
    return $flash;
}

/**
 * Envoie une réponse JSON propre
 *
 * @param mixed $donnees  Données à encoder
 * @param int   $code     Code HTTP
 */
function repondreJSON(mixed $donnees, int $code = 200): void
{
    http_response_code($code);
    header('Content-Type: application/json; charset=UTF-8');
    header('Cache-Control: no-store');
    echo json_encode($donnees, JSON_UNESCAPED_UNICODE | JSON_PRETTY_PRINT);
    exit;
}
```

### Utilisation des messages flash

```php

```

### **`src/helpers/http.php`**

```php
<?php
declare(strict_types=1);

/**
 * http.php
 * Helpers HTTP — redirections, flash messages, JSON
 */

// ============================================================
// REDIRECTIONS
// ============================================================

/**
 * Redirige vers une URL avec un code HTTP
 * Toujours appelé avec exit — stoppe le script
 *
 * @param string $url  URL de destination
 * @param int    $code Code HTTP (303 = See Other après POST)
 */
function rediriger(string $url, int $code = 303): void
{
    if (headers_sent()) {
        // Fallback si headers déjà envoyés — rare mais possible
        echo "<script>window.location.href='" . addslashes($url) . "'</script>";
        exit;
    }

    header('Location: ' . $url, true, $code);
    exit;
}

/**
 * Redirige avec un message flash stocké en session
 * Le message sera affiché une seule fois sur la page suivante
 *
 * @param string $url     URL de destination
 * @param string $message Message à afficher
 * @param string $type    'succes' | 'erreur' | 'info' | 'attention'
 */
function redigirerAvecMessage(
    string $url,
    string $message,
    string $type = 'succes'
): void {
    $_SESSION['flash'] = [
        'message' => $message,
        'type'    => $type
    ];
    rediriger($url);
}

// ============================================================
// MESSAGES FLASH
// ============================================================

/**
 * Récupère le message flash et le supprime de la session
 * One-shot — lu une fois puis effacé
 *
 * @return array|null ['message' => '...', 'type' => '...'] ou null
 */
function getMessageFlash(): ?array
{
    if (!isset($_SESSION['flash'])) {
        return null;
    }

    $flash = $_SESSION['flash'];
    unset($_SESSION['flash']);   // Supprimer après lecture
    return $flash;
}

/**
 * Affiche le message flash s'il existe
 * À appeler dans le template — header.php ou en haut de chaque page
 */
function afficherMessageFlash(): void
{
    $flash = getMessageFlash();

    if ($flash === null) {
        return;
    }

    // Couleurs selon le type
    $styles = [
        'succes'    => 'background:#d4edda; color:#155724; border:1px solid #c3e6cb',
        'erreur'    => 'background:#f8d7da; color:#721c24; border:1px solid #f5c6cb',
        'info'      => 'background:#d1ecf1; color:#0c5460; border:1px solid #bee5eb',
        'attention' => 'background:#fff3cd; color:#856404; border:1px solid #ffeeba',
    ];

    $style = $styles[$flash['type']] ?? $styles['info'];

    echo sprintf(
        '<div style="%s; padding:10px; margin:10px 0; border-radius:4px">%s</div>',
        $style,
        escape($flash['message'])
    );
}

// ============================================================
// JSON
// ============================================================

/**
 * Envoie une réponse JSON et stoppe le script
 * Définit automatiquement Content-Type et le code HTTP
 *
 * @param mixed $donnees Données à encoder en JSON
 * @param int   $code    Code HTTP de la réponse
 */
function repondreJSON(mixed $donnees, int $code = 200): void
{
    // Pas de cache sur les réponses API
    header('Cache-Control: no-store');
    header('Content-Type: application/json; charset=UTF-8');

    http_response_code($code);

    echo json_encode(
        $donnees,
        JSON_UNESCAPED_UNICODE |   // Garde les accents
        JSON_PRETTY_PRINT      |   // Formatage lisible
        JSON_UNESCAPED_SLASHES     // Garde les slashes
    );
    exit;
}
```

### **`public/api.php`**

```php
<?php
declare(strict_types=1);

/**
 * api.php
 * Mini API JSON — articles
 *
 * GET  /api.php?ressource=articles         → liste tous les articles
 * GET  /api.php?ressource=articles&id=1    → un article par ID
 * POST /api.php?ressource=articles         → créer un article
 * PUT  /api.php?ressource=articles&id=1    → modifier un article
 * DELETE /api.php?ressource=articles&id=1  → supprimer un article
 */

require_once __DIR__ . '/../src/config/config.php';
require_once __DIR__ . '/../src/helpers/errors.php';
require_once __DIR__ . '/../src/helpers/security.php';
require_once __DIR__ . '/../src/helpers/functions.php';
require_once __DIR__ . '/../src/helpers/http.php';
require_once __DIR__ . '/../src/config/database.php';
require_once __DIR__ . '/../src/helpers/crud.php';

// Toujours JSON pour cette API
header('Content-Type: application/json; charset=UTF-8');
header('Cache-Control: no-store');

// Récupérer la méthode et les paramètres
$methode   = $_SERVER['REQUEST_METHOD'];
$ressource = preg_replace('/[^a-z_]/', '', $_GET['ressource'] ?? '');
$id        = (int)($_GET['id'] ?? 0);

// Ressources autorisées — liste blanche
$ressourcesAutorisees = ['articles'];

if (!in_array($ressource, $ressourcesAutorisees, true)) {
    repondreJSON([
        'succes' => false,
        'erreur' => "Ressource inconnue : {$ressource}"
    ], 404);
}

// ============================================================
// ROUTAGE API
// ============================================================

try {

    // ── GET — liste ou détail ────────────────────────────────

    if ($methode === 'GET') {

        if ($id > 0) {
            // GET ?ressource=articles&id=1 → un article
            $article = fetchById($ressource, $id);

            repondreJSON([
                'succes'  => true,
                'article' => $article
            ]);

        } else {
            // GET ?ressource=articles → tous les articles
            $page    = max(1, (int)($_GET['p'] ?? 1));
            $parPage = min(50, max(1, (int)($_GET['par_page'] ?? 10)));

            $result = fetchAll($ressource, 'created_at', $page, $parPage);

            repondreJSON([
                'succes'        => true,
                'total'         => $result['total'],
                'pages'         => $result['pages'],
                'page_actuelle' => $result['page_actuelle'],
                'articles'      => $result['data']
            ]);
        }
    }

    // ── POST — créer ─────────────────────────────────────────

    if ($methode === 'POST') {

        // Lire le body JSON
        $body = json_decode(file_get_contents('php://input'), true);

        if (!is_array($body)) {
            repondreJSON([
                'succes' => false,
                'erreur' => 'Body JSON invalide ou manquant'
            ], 400);
        }

        $titre   = trim($body['titre']   ?? '');
        $contenu = trim($body['contenu'] ?? '');
        $auteur  = trim($body['auteur']  ?? '');
        $statut  = $body['statut']       ?? 'brouillon';

        // Validation
        $erreurs = [];
        if (empty($titre))   $erreurs[] = "Le titre est obligatoire";
        if (empty($contenu)) $erreurs[] = "Le contenu est obligatoire";
        if (empty($auteur))  $erreurs[] = "L'auteur est obligatoire";
        if (!in_array($statut, ['publie', 'brouillon'], true)) {
            $erreurs[] = "Statut invalide";
        }

        if (!empty($erreurs)) {
            repondreJSON([
                'succes'  => false,
                'erreurs' => $erreurs
            ], 422);
        }

        $newId = insert($ressource, [
            'titre'   => $titre,
            'contenu' => $contenu,
            'auteur'  => $auteur,
            'statut'  => $statut,
        ]);

        repondreJSON([
            'succes'  => true,
            'message' => 'Article créé avec succès',
            'id'      => $newId
        ], 201);
    }

    // ── PUT — modifier ───────────────────────────────────────

    if ($methode === 'PUT') {

        if ($id <= 0) {
            repondreJSON([
                'succes' => false,
                'erreur' => 'ID manquant ou invalide'
            ], 400);
        }

        $body = json_decode(file_get_contents('php://input'), true);

        if (!is_array($body)) {
            repondreJSON([
                'succes' => false,
                'erreur' => 'Body JSON invalide'
            ], 400);
        }

        // Récupérer l'article existant pour ne modifier que les champs fournis
        $articleExistant = fetchById($ressource, $id);

        $donnees = [
            'titre'   => trim($body['titre']   ?? $articleExistant['titre']),
            'contenu' => trim($body['contenu'] ?? $articleExistant['contenu']),
            'auteur'  => trim($body['auteur']  ?? $articleExistant['auteur']),
            'statut'  => $body['statut']       ?? $articleExistant['statut'],
        ];

        if (!in_array($donnees['statut'], ['publie', 'brouillon'], true)) {
            $donnees['statut'] = 'brouillon';
        }

        update($ressource, $id, $donnees);

        repondreJSON([
            'succes'  => true,
            'message' => "Article {$id} modifié avec succès"
        ]);
    }

    // ── DELETE — supprimer ───────────────────────────────────

    if ($methode === 'DELETE') {

        if ($id <= 0) {
            repondreJSON([
                'succes' => false,
                'erreur' => 'ID manquant ou invalide'
            ], 400);
        }

        delete($ressource, $id);

        repondreJSON([
            'succes'  => true,
            'message' => "Article {$id} supprimé avec succès"
        ]);
    }

    // Méthode non supportée
    repondreJSON([
        'succes' => false,
        'erreur' => "Méthode {$methode} non supportée"
    ], 405);

} catch (NotFoundException $e) {
    repondreJSON(['succes' => false, 'erreur' => $e->getMessage()], 404);
} catch (ValidationException $e) {
    repondreJSON(['succes' => false, 'erreur' => $e->getMessage()], 400);
} catch (PDOException $e) {
    loggerErreur($e);
    repondreJSON(['succes' => false, 'erreur' => 'Erreur serveur'], 500);
} catch (Exception $e) {
    loggerErreur($e);
    repondreJSON(['succes' => false, 'erreur' => 'Erreur inattendue'], 500);
}
```

### **Comment tester l'API**

```bash
# GET — tous les articles
curl http://localhost/agence/public/api.php?ressource=articles

# GET — un article
curl http://localhost/agence/public/api.php?ressource=articles&id=1

# POST — créer un article
curl -X POST http://localhost/agence/public/api.php?ressource=articles \
  -H "Content-Type: application/json" \
  -d '{"titre":"Mon article","contenu":"Contenu ici","auteur":"Jack","statut":"publie"}'

# PUT — modifier
curl -X PUT "http://localhost/agence/public/api.php?ressource=articles&id=1" \
  -H "Content-Type: application/json" \
  -d '{"titre":"Titre modifié"}'

# DELETE — supprimer
curl -X DELETE "http://localhost/agence/public/api.php?ressource=articles&id=1"
```
