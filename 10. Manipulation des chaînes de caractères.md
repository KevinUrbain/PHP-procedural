# Manipulation des chaînes de caractères

# Étape 7 — Manipulation des chaînes de caractères

## 1. La théorie complète

### Les deux types de chaînes

```php
// Quotes simples — texte brut, pas d'interpolation
$nom = 'Jack';
echo 'Bonjour $nom';     // Affiche : Bonjour $nom

// Quotes doubles — interpolation des variables
$nom = 'Jack';
echo "Bonjour $nom";     // Affiche : Bonjour Jack
echo "Bonjour {$nom}";   // Affiche : Bonjour Jack (recommandé)
```

### **Concaténation et interpolation**

```php
$prenom = "Jack";
$nom    = "Jones";

// Concaténation avec .
$nomComplet = $prenom . " " . $nom;

// Concaténation avec .=
$message  = "Bonjour ";
$message .= $prenom;

// Interpolation — plus lisible quand il y a plusieurs variables
$message = "Bonjour {$prenom} {$nom}, bienvenue !";
```

### Les fonctions essentielles

### Longueur et casse

```php
$texte = "  Bonjour le Monde  ";

// Longueur
echo strlen($texte);          // 20 — compte les espaces aussi

// Casse
echo strtolower($texte);      // "  bonjour le monde  "
echo strtoupper($texte);      // "  BONJOUR LE MONDE  "
echo ucfirst("bonjour");      // "Bonjour" — 1ère lettre en majuscule
echo ucwords("bonjour monde");// "Bonjour Monde" — 1ère lettre de chaque mot
```

### **Nettoyage**

```php
$texte = "  Bonjour le Monde  ";

// Supprimer les espaces en début et fin
echo trim($texte);            // "Bonjour le Monde"
echo ltrim($texte);           // "Bonjour le Monde  " — gauche seulement
echo rtrim($texte);           // "  Bonjour le Monde" — droite seulement

// Supprimer des caractères spécifiques
echo trim("***Bonjour***", "*");  // "Bonjour"
```

### **Recherche**

```php
$texte = "Bonjour le monde";

// Trouver la position d'une sous-chaîne
$position = strpos($texte, "monde");
var_dump($position);   // int(11)

// Retourne false si non trouvé — toujours vérifier avec ===
$position = strpos($texte, "xyz");
var_dump($position);   // bool(false)

// ❌ Piège classique
if ($position == false) { }   // 0 == false est TRUE — bug !

// ✅ Toujours comparer avec ===
if ($position === false) { }

// Vérifier si une chaîne contient une sous-chaîne (PHP 8+)
var_dump(str_contains($texte, "monde"));    // bool(true)
var_dump(str_starts_with($texte, "Bon"));   // bool(true)
var_dump(str_ends_with($texte, "monde"));   // bool(true)
```

### **Extraction**

```php
$texte = "Bonjour le monde";

// substr(texte, début, longueur)
echo substr($texte, 0, 7);    // "Bonjour" — 7 caractères depuis le début
echo substr($texte, 8);       // "le monde" — depuis la position 8 jusqu'à la fin
echo substr($texte, -5);      // "monde" — 5 derniers caractères
```

### **Remplacement**

```php
$texte = "Bonjour le monde";

// str_replace(chercher, remplacer, dans)
echo str_replace("monde", "PHP", $texte);
// "Bonjour le PHP"

// Remplacement multiple — tableaux
$texte = "Le chat mange la souris";
echo str_replace(
    ["chat", "souris"],
    ["chien", "balle"],
    $texte
);
// "Le chien mange la balle"

// Insensible à la casse
echo str_ireplace("MONDE", "PHP", "Bonjour le monde");
// "Bonjour le PHP"
```

### **Découper et assembler**

```php
// explode — découpe une chaîne en tableau
$csv   = "Jack,Alice,Bob,Sara";
$noms  = explode(",", $csv);
var_dump($noms);
// ["Jack", "Alice", "Bob", "Sara"]

// Avec limite — arrête après N morceaux
$noms = explode(",", $csv, 2);
// ["Jack", "Alice,Bob,Sara"]

// implode — assemble un tableau en chaîne
$noms    = ["Jack", "Alice", "Bob"];
$resultat = implode(", ", $noms);
echo $resultat;   // "Jack, Alice, Bob"

// Cas concret — générer une liste HTML
$tags = ["PHP", "MySQL", "HTML"];
echo implode(" | ", $tags);   // "PHP | MySQL | HTML"
```

### **Formatage**

```php
// sprintf — formate une chaîne avec des variables
$prenom = "Jack";
$solde  = 1234.5;

$message = sprintf("Bonjour %s, votre solde est de %.2f €", $prenom, $solde);
echo $message;
// "Bonjour Jack, votre solde est de 1234.50 €"

// Formats courants
// %s  → string
// %d  → entier
// %f  → float
// %.2f → float avec 2 décimales
// %05d → entier sur 5 chiffres avec zéros : 00042

// Générer une référence de commande
$id  = 42;
$ref = sprintf("CMD-%05d", $id);
echo $ref;   // "CMD-00042"
```

### **Répétition et padding**

```php
// str_repeat — répéter une chaîne
echo str_repeat("=", 20);   // "===================="
echo str_repeat("* ", 3);   // "* * * "

// str_pad — compléter une chaîne jusqu'à une longueur
echo str_pad("42", 5, "0", STR_PAD_LEFT);    // "00042"
echo str_pad("PHP", 10, "-");                 // "PHP-------"
echo str_pad("PHP", 10, "-", STR_PAD_BOTH);  // "---PHP----"
```

### **Conversion et nettoyage sécurisé**

```php
// htmlspecialchars — INDISPENSABLE avant tout affichage utilisateur
$input = "<script>alert('XSS')</script>";
echo htmlspecialchars($input, ENT_QUOTES, 'UTF-8');
// "&lt;script&gt;alert(&#039;XSS&#039;)&lt;/script&gt;"
// Le script est neutralisé — affiché comme texte, pas exécuté

// strip_tags — supprime toutes les balises HTML
$html = "<p>Bonjour <strong>le monde</strong></p>";
echo strip_tags($html);          // "Bonjour le monde"

// Autoriser certaines balises
echo strip_tags($html, "<strong>");  // "Bonjour <strong>le monde</strong>"

// nl2br — convertit les sauts de ligne en <br>
$texte = "Ligne 1\nLigne 2\nLigne 3";
echo nl2br($texte);
// "Ligne 1<br>\nLigne 2<br>\nLigne 3"
```

### **Fonctions PHP 8 indispensables**

```php
// str_contains — contient une sous-chaîne ?
str_contains("Bonjour le monde", "monde");    // true

// str_starts_with — commence par ?
str_starts_with("Bonjour", "Bon");            // true

// str_ends_with — finit par ?
str_ends_with("Bonjour", "jour");             // true
```

## 2. Exemple concret et utile

Un **système de traitement de données utilisateur** — cas classique en entreprise :

```php
<?php
declare(strict_types=1);

/**
 * utilisateurs.php
 * Nettoyage et formatage des données utilisateur
 */

// Données brutes reçues d'un formulaire (on verra $_POST plus tard)
$donneesbrutes = [
    ["nom" => "  dupont  ",   "email" => "DUPONT@GMAIL.COM",   "bio" => "<script>alert('xss')</script>Dev PHP"],
    ["nom" => "MARTIN",       "email" => "martin@outlook.com",  "bio" => "  Passionné de   web  "],
    ["nom" => "bernard",      "email" => "BERNARD@YAHOO.FR",    "bio" => "Designer <b>UI/UX</b>"],
];

/**
 * Nettoie et formate les données d'un utilisateur
 *
 * @param array $user Données brutes
 * @return array      Données nettoyées
 */
function normaliserUtilisateur(array $user): array
{
    // Nom — trim + ucwords pour une capitalisation propre
    $user['nom'] = ucwords(strtolower(trim($user['nom'])));

    // Email — trim + strtolower (les emails sont insensibles à la casse)
    $user['email'] = strtolower(trim($user['email']));

    // Bio — strip_tags pour supprimer le HTML + trim des espaces
    $user['bio'] = trim(strip_tags($user['bio']));

    // Générer un identifiant unique à partir du nom
    $user['identifiant'] = strtolower(str_replace(" ", "_", $user['nom']));

    return $user;
}

/**
 * Génère un résumé formaté de l'utilisateur
 *
 * @param array $user Utilisateur normalisé
 * @return string     Résumé formaté
 */
function genererResume(array $user): string
{
    return sprintf(
        "[%s] %s (%s) — %s",
        strtoupper(substr($user['nom'], 0, 3)),  // 3 premières lettres en majuscules
        $user['nom'],
        $user['email'],
        $user['bio']
    );
}

// Traitement
$utilisateurs = array_map('normaliserUtilisateur', $donneesbrutes);

// Affichage
foreach ($utilisateurs as $user) {
    echo genererResume($user) . "<br>";
}

// Résultat :
// [DUP] Dupont (dupont@gmail.com) — Dev PHP
// [MAR] Martin (martin@outlook.com) — Passionné de   web
// [BER] Bernard (bernard@yahoo.fr) — Designer UI/UX
```

## **3. Bonnes pratiques et erreurs à éviter**

```php
// ❌ Afficher du contenu utilisateur sans protection
echo $user['commentaire'];   // Faille XSS !

// ✅ Toujours échapper avant affichage
echo htmlspecialchars($user['commentaire'], ENT_QUOTES, 'UTF-8');

// ❌ Utiliser == avec strpos
if (strpos($texte, "php") == false) { }   // 0 == false → bug !

// ✅ Toujours === avec strpos
if (strpos($texte, "php") === false) { }

// ❌ Concaténation lourde
$html  = "<tr>";
$html .= "<td>";
$html .= $nom;
$html .= "</td>";
$html .= "</tr>";

// ✅ sprintf pour les templates courts
$html = sprintf("<tr><td>%s</td></tr>", htmlspecialchars($nom, ENT_QUOTES, 'UTF-8'));

// ❌ Oublier trim sur les données utilisateur
$email = $_POST['email'];   // "  jack@gmail.com  " — espaces parasites

// ✅ Toujours trim en premier
$email = trim($_POST['email']);
```

## 4. Ce que font les seniors 👨‍💻

Les seniors créent des **fonctions utilitaires réutilisables** qu'ils mettent dans un fichier `helpers/strings.php` :

```php
<?php
// helpers/strings.php
declare(strict_types=1);

/**
 * Nettoie une chaîne pour affichage sécurisé
 *
 * @param string $valeur Valeur brute
 * @return string        Valeur nettoyée et sécurisée
 */
function nettoyerChaine(string $valeur): string
{
    return htmlspecialchars(trim($valeur), ENT_QUOTES, 'UTF-8');
}

/**
 * Génère un slug à partir d'un titre
 * Ex: "Mon Article PHP" → "mon-article-php"
 *
 * @param string $titre Titre brut
 * @return string       Slug formaté
 */
function genererSlug(string $titre): string
{
    $slug = strtolower(trim($titre));
    $slug = str_replace(
        ['à','â','é','è','ê','ë','î','ï','ô','ù','û','ü','ç',' '],
        ['a','a','e','e','e','e','i','i','o','u','u','u','c','-'],
        $slug
    );
    return $slug;
}

/**
 * Tronque un texte à une longueur donnée
 *
 * @param string $texte     Texte à tronquer
 * @param int    $longueur  Longueur maximale
 * @param string $suffixe   Suffixe ajouté si tronqué
 * @return string           Texte tronqué
 */
function tronquer(string $texte, int $longueur = 100, string $suffixe = '...'): string
{
    if (strlen($texte) <= $longueur) {
        return $texte;
    }
    return substr($texte, 0, $longueur) . $suffixe;
}
```

    - Normalise tous les articles avec `array_map`

/**
 * Normalise les données brutes d'un article
 *
 * @param array $article Article brut
 * @return array         Article normalisé
 */
function normaliserArticle(array $article): array
{
    // TITRE
    // 1. trim  — supprime les espaces en début/fin
    // 2. strtolower — tout en minuscules
    // 3. ucfirst — première lettre en majuscule
    // Ordre crucial : strtolower AVANT ucfirst
    $article['titre'] = ucfirst(strtolower(trim($article['titre'])));

    // CONTENU
    // 1. strip_tags — supprime toutes les balises HTML (dont le script XSS)
    // 2. trim — supprime les espaces parasites
    $article['contenu'] = trim(strip_tags($article['contenu']));

    // AUTEUR
    // 1. trim — espaces
    // 2. strtolower — tout en minuscules d'abord
    // 3. ucwords — première lettre de chaque mot en majuscule
    // Ordre crucial : strtolower AVANT ucwords
    $article['auteur'] = ucwords(strtolower(trim($article['auteur'])));

    // SLUG — généré à partir du titre déjà nettoyé
    // On remplace les accents pour avoir un slug ASCII pur
    $slug = strtolower(trim($article['titre']));
    $slug = str_replace(
        ['à','â','é','è','ê','î','ï','ô','ù','û','ü','ç',' '],
        ['a','a','e','e','e','i','i','o','u','u','u','c','-'],
        $slug
    );
    $article['slug'] = $slug;

    // TAGS — explode convertit la chaîne "php,web" en tableau ["php","web"]
    $article['tags'] = explode(',', $article['tags']);

    // NOMBRE DE MOTS — sur le contenu déjà nettoyé
    $article['nbMots'] = str_word_count($article['contenu']);

    return $article;
}

/**
 * Génère un extrait tronqué du contenu
 *
 * @param string $contenu  Contenu de l'article
 * @param int    $longueur Nombre maximum de caractères
 * @return string          Extrait formaté
 */
function genererExtrait(string $contenu, int $longueur = 100): string
{
    // Guard clause — si le contenu est assez court, on retourne tel quel
    if (strlen($contenu) <= $longueur) {
        return $contenu;
    }

    // Sinon on tronque à $longueur caractères et on ajoute "..."
    return substr($contenu, 0, $longueur) . '...';
}

/**
 * Formate un nombre de vues en notation lisible
 *
 * @param int $vues Nombre de vues brut

 */
function formaterVues(int $vues): string
{

        return sprintf("%.1fk", $vues / 1000);
        // 1542 / 1000 = 1.542 → "1.5k"
        // 2103 / 1000 = 2.103 → "2.1k"
    }

    return (string) $vues;   // 892 → "892"
}

# Bilan

| Critère | Résultat |
|---|---|
| `normaliserArticle()` — structure | ✅ |
| Ordre `strtolower` → `ucfirst` | ❌ Inversé |
| Ordre `strtolower` → `ucwords` | ❌ Inversé |
| Slug sans accents | 🔧 Ajouté |
| `explode` pour les tags | ✅ |
| `str_word_count` | ✅ |
| `genererExtrait()` — `strlen` vs `str_word_count` | ❌ Bug corrigé |

# Le point le plus important à retenir
```
┌──────────────────────────────────────────────────────┐
│  L'ordre des fonctions imbriquées est crucial        │
│  On lit de l'intérieur vers l'extérieur              │
│                                                      │
│  ucfirst(strtolower(trim($texte)))                   │
│          1. trim      → supprime les espaces         │
│       2. strtolower   → tout en minuscules           │
│  3. ucfirst           → 1ère lettre en majuscule     │
│                                                      │
│  Inverser cet ordre = résultat incorrect             │
└──────────────────────────────────────────────────────┘
```
