# Exercice supplémentaire callbacks

# 🎯 Exercice — Callbacks et fonctions de tableaux

Un exercice concret, sur un cas réel que tu peux rencontrer en entreprise.

---

## Le contexte

Tu travailles pour une **boutique en ligne**. Tu reçois une liste de commandes à traiter.

```php
<?php
declare(strict_types=1);

$commandes = [
    ["id" => 1, "client" => "Dupont",  "produit" => "Laptop",  "quantite" => 2, "prix_unitaire" => 899.0,  "statut" => "en_attente", "membre_vip" => false],
    ["id" => 2, "client" => "Martin",  "produit" => "Souris",  "quantite" => 5, "prix_unitaire" => 29.0,   "statut" => "expediee",   "membre_vip" => true],
    ["id" => 3, "client" => "Bernard", "produit" => "Clavier", "quantite" => 1, "prix_unitaire" => 79.0,   "statut" => "en_attente", "membre_vip" => true],
    ["id" => 4, "client" => "Leroy",   "produit" => "Écran",   "quantite" => 3, "prix_unitaire" => 349.0,  "statut" => "annulee",    "membre_vip" => false],
    ["id" => 5, "client" => "Moreau",  "produit" => "Webcam",  "quantite" => 2, "prix_unitaire" => 59.0,   "statut" => "en_attente", "membre_vip" => true],
    ["id" => 6, "client" => "Simon",   "produit" => "Casque",  "quantite" => 1, "prix_unitaire" => 89.0,   "statut" => "expediee",   "membre_vip" => false],
];

$tva             = 0.21;
$remiseVip       = 0.10;  // 10% de remise pour les membres VIP
$seuilLivraison  = 100.0; // Livraison gratuite au-dessus de 100€
$fraisLivraison  = 9.99;  // Frais de livraison si en-dessous du seuil
```

## Ce que tu dois faire

**Uniquement** avec `array_filter`, `array_map`, `array_reduce`, `usort` et des callbacks :

### 1. Filtrer

Garde uniquement les commandes dont le statut est `"en_attente"` ou `"expediee"` — pas les annulées.

### 2. Enrichir avec `array_map`

Pour chaque commande restante, calcule et ajoute :

- `sous_total` : `prix_unitaire * quantite`
- `remise` : si `membre_vip === true` → `sous_total * remiseVip`, sinon `0.0`
- `sous_total_apres_remise` : `sous_total - remise`
- `frais_livraison` : si `sous_total_apres_remise >= seuilLivraison` → `0.0`, sinon `$fraisLivraison`
- `montant_ttc` : `(sous_total_apres_remise + frais_livraison) * (1 + tva)`

### 3. Trier

Par `montant_ttc` **décroissant** avec `usort`.

### 4. Calculer avec `array_reduce`

- Le **total TTC** de toutes les commandes
- Le **nombre de commandes VIP** — oui, `array_reduce` peut aussi compter !

### 5. Afficher

Avec `var_dump()` :

- Le tableau final des commandes enrichies
- Le total TTC
- Le nombre de commandes VIP

---

## Contraintes

- `use()` obligatoire dans au moins un callback
- Arrow function `fn` dans au moins un callback
- Pas de boucle `foreach` — uniquement les fonctions natives

```php
<?php
declare(strict_types=1);

/**
 * commandes.php
 * Traitement des commandes d'une boutique en ligne
 */

// ============================================================
// DONNÉES
// ============================================================

$commandes = [
    ["id" => 1, "client" => "Dupont",  "produit" => "Laptop",  "quantite" => 2, "prix_unitaire" => 899.0, "statut" => "en_attente", "membre_vip" => false],
    ["id" => 2, "client" => "Martin",  "produit" => "Souris",  "quantite" => 5, "prix_unitaire" => 29.0,  "statut" => "expediee",   "membre_vip" => true],
    ["id" => 3, "client" => "Bernard", "produit" => "Clavier", "quantite" => 1, "prix_unitaire" => 79.0,  "statut" => "en_attente", "membre_vip" => true],
    ["id" => 4, "client" => "Leroy",   "produit" => "Écran",   "quantite" => 3, "prix_unitaire" => 349.0, "statut" => "annulee",    "membre_vip" => false],
    ["id" => 5, "client" => "Moreau",  "produit" => "Webcam",  "quantite" => 2, "prix_unitaire" => 59.0,  "statut" => "en_attente", "membre_vip" => true],
    ["id" => 6, "client" => "Simon",   "produit" => "Casque",  "quantite" => 1, "prix_unitaire" => 89.0,  "statut" => "expediee",   "membre_vip" => false],
];

$tva            = 0.21;
$remiseVip      = 0.10;
$seuilLivraison = 100.0;
$fraisLivraison = 9.99;

// ============================================================
// ÉTAPE 1 — Filtrer les commandes actives (pas les annulées)
// Arrow function suffisante — condition simple, une ligne
// array_values() pour réindexer proprement
// ============================================================

$commandesFiltrees = array_values(array_filter(
    $commandes,
    fn(array $c): bool => $c['statut'] === 'en_attente' || $c['statut'] === 'expediee'
));

// ============================================================
// ÉTAPE 2 — Enrichir chaque commande
// Fonction anonyme — plusieurs lignes + variables externes
// use() obligatoire pour importer les variables du scope parent
// ============================================================

$commandesEnrichies = array_map(
    function(array $c) use ($remiseVip, $seuilLivraison, $fraisLivraison, $tva): array {

        // Sous-total brut
        $c['sous_total'] = $c['prix_unitaire'] * $c['quantite'];

        // Remise VIP — montant calculé puis soustrait
        $c['remise'] = $c['membre_vip'] ? $c['sous_total'] * $remiseVip : 0.0;

        // Sous-total après remise — on soustrait le MONTANT, pas le taux
        $c['sous_total_apres_remise'] = $c['sous_total'] - $c['remise'];

        // Frais de livraison selon le seuil
        $c['frais_livraison'] = $c['sous_total_apres_remise'] >= $seuilLivraison
            ? 0.0
            : $fraisLivraison;

        // Montant TTC — on ADDITIONNE sous_total + frais, puis on applique la TVA
        $c['montant_ttc'] = round(
            ($c['sous_total_apres_remise'] + $c['frais_livraison']) * (1 + $tva),
            2
        );

        return $c;
    },
    $commandesFiltrees
);

// ============================================================
// ÉTAPE 3 — Trier par montant TTC décroissant
// $b <=> $a pour décroissant
// ============================================================

usort(
    $commandesEnrichies,
    fn(array $a, array $b): int => $b['montant_ttc'] <=> $a['montant_ttc']
);

// ============================================================
// ÉTAPE 4 — Calculer le total TTC et compter les VIP
// array_reduce peut accumuler N'IMPORTE QUELLE valeur
// ============================================================

$totalTTC = array_reduce(
    $commandesEnrichies,
    fn(float $total, array $c): float => $total + $c['montant_ttc'],
    0.0
);

$nbCommandesVip = array_reduce(
    $commandesEnrichies,
    fn(int $compteur, array $c): int => $compteur + ($c['membre_vip'] ? 1 : 0),
    0
);

// ============================================================
// AFFICHAGE
// ============================================================

echo "<pre>";
var_dump($commandesEnrichies);
var_dump($totalTTC);
var_dump($nbCommandesVip);
echo "</pre>";
```

---

## Ce que `var_dump` doit afficher
```
Commandes triées par montant_ttc décroissant :

1. Dupont  — Laptop  x2 — 2 179,58 €  (1798 + 0 livraison) * 1.21
2. Moreau  — Webcam  x2 —   141,50 €  (118 - remise VIP 11.8 + 0 livraison) * 1.21
3. Martin  — Souris  x5 —   169,07 €  (145 - remise VIP 14.5 + 9.99 livraison) * 1.21
4. Bernard — Clavier x1 —    96,41 €  (79 - remise VIP 7.9 + 9.99 livraison) * 1.21
5. Simon   — Casque  x1 —   119,96 €  (89 + 9.99 livraison) * 1.21

float(...)   ← total TTC
int(3)       ← Martin, Bernard, Moreau sont VIP
```

---

## Bilan

| Critère | Résultat |
|---|---|
| `array_filter` avec condition combinée | ✅ |
| `array_values()` après `array_filter` | ❌ Manquant |
| `array_map` avec `use()` | ✅ |
| `$f['remise']` au lieu de `$remiseVip` | ❌ Bug corrigé |
| `+` au lieu de `*` pour `montant_ttc` | ❌ Bug corrigé |
| `usort` décroissant | ✅ |
| `array_reduce` pour le total | ✅ |
| `array_reduce` pour compter les VIP | ❌ Manquant |

## Le point le plus important à retenir
```
┌──────────────────────────────────────────────────────┐
│  Quand tu calcules en plusieurs étapes               │
│  utilise TOUJOURS la valeur calculée                 │
│  pas la variable de configuration                    │
│                                                      │
│  ❌ $f['sous_total'] - $remiseVip   (le taux 0.10)  │
│  ✅ $f['sous_total'] - $f['remise'] (le montant)    │
└──────────────────────────────────────────────────────┘
```
