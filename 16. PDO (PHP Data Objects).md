## 1. La théorie complète

### C'est quoi PDO ?

PDO est une **couche d'abstraction** pour accéder aux bases de données. Contrairement à MySQLi qui ne fonctionne qu'avec MySQL, PDO supporte **12 moteurs de bases de données** avec la même syntaxe.

```
MySQLi → uniquement MySQL
PDO    → MySQL, PostgreSQL, SQLite, Oracle, SQL Server...

Si tu changes de base de données → tu changes juste la connexion
Le reste du code reste identique ✅
```

### **MySQLi vs PDO — comparaison directe**

```php
// ══ MYSQLI ══════════════════════════════════════════════
$stmt = mysqli_prepare($co, "SELECT * FROM users WHERE id = ?");
mysqli_stmt_bind_param($stmt, "i", $id);
mysqli_stmt_execute($stmt);
$result = mysqli_stmt_get_result($stmt);
$user   = mysqli_fetch_assoc($result);

// ══ PDO ═════════════════════════════════════════════════
$stmt = $pdo->prepare("SELECT * FROM users WHERE id = ?");
$stmt->execute([$id]);
$user = $stmt->fetch(PDO::FETCH_ASSOC);

// PDO est plus concis et orienté objet
// Même si on est en procédural, PDO utilise des objets
```

### **Connexion PDO**

```php
<?php
declare(strict_types=1);

// PDO utilise un DSN — Data Source Name
// "mysql:host=localhost;dbname=ma_base;port=port;charset=utf8mb4"
// Le port est optionnel selon si on en a besoin ou non

try {

    $pdo = new PDO(
        "mysql:host=localhost;dbname=ma_base;charset=utf8mb4",
        "root",       // utilisateur
        "",           // mot de passe
        [
            // Options PDO — très importantes
            PDO::ATTR_ERRMODE            => PDO::ERRMODE_EXCEPTION,
            // ↑ Lance une PDOException en cas d'erreur SQL

            PDO::ATTR_DEFAULT_FETCH_MODE => PDO::FETCH_ASSOC,
            // ↑ fetch() retourne toujours un tableau associatif

            PDO::ATTR_EMULATE_PREPARES   => false,
            // ↑ Utilise les vraies requêtes préparées du moteur DB
        ]
    );

} catch (PDOException $e) {
    // Ne jamais afficher $e->getMessage() en production
    // Ça peut révéler les credentials de la DB !
    die("Connexion impossible");
}
```

### **Les modes de fetch**

```php
// FETCH_ASSOC — tableau associatif (le plus utilisé)
$user = $stmt->fetch(PDO::FETCH_ASSOC);
// ["id" => 1, "prenom" => "Jack", "email" => "jack@test.com"]

// FETCH_NUM — tableau indexé
$user = $stmt->fetch(PDO::FETCH_NUM);
// [1, "Jack", "jack@test.com"]

// FETCH_OBJ — objet stdClass
$user = $stmt->fetch(PDO::FETCH_OBJ);
// $user->prenom, $user->email

// fetchAll — toutes les lignes
$users = $stmt->fetchAll(PDO::FETCH_ASSOC);
// [[...], [...], [...]]
```

### **READ — Lire des données**

```php

$stmt->execute();

$article = $stmt->fetch();

    throw new NotFoundException("Article {$id} introuvable");

// ── Requête avec plusieurs paramètres ───────────────────

$stmt = $pdo->prepare("
    SELECT * FROM articles
    WHERE statut = ?
    AND auteur = ?
    ORDER BY created_at DESC
");
$stmt->execute([$statut, $auteur]);
$articles = $stmt->fetchAll();

// ── Paramètres nommés — plus lisible ────────────────────

$stmt = $pdo->prepare("
    SELECT * FROM articles
    WHERE statut = :statut
    AND auteur = :auteur
");
$stmt->execute([
    ':statut' => $statut,
    ':auteur' => $auteur,

$articles = $stmt->fetchAll();
```

### Les transactions

Une transaction garantit que **plusieurs opérations réussissent ensemble** ou échouent ensemble :

```php
// Exemple : créer un utilisateur ET son profil
// Les deux doivent réussir — sinon on annule tout

try {

    $pdo->beginTransaction();   // Début de la transaction

    // Opération 1 — créer l'utilisateur
    $stmt = $pdo->prepare("INSERT INTO utilisateurs (email) VALUES (?)");
    $stmt->execute([$email]);
    $userId = (int)$pdo->lastInsertId();

    // Opération 2 — créer son profil
    $stmt = $pdo->prepare("INSERT INTO profils (user_id, bio) VALUES (?, ?)");
    $stmt->execute([$userId, $bio]);

    // Tout s'est bien passé — on valide
    $pdo->commit();

    echo "Utilisateur et profil créés !";

} catch (PDOException $e) {
    // Une erreur — on annule TOUT
    $pdo->rollBack();
    loggerErreur($e);
    throw new DatabaseException("Création échouée", 500);
}
```

## 2. Exemple concret — fichier de connexion PDO

### `src/config/database.php` — version PDO

```php
<?php
declare(strict_types=1);

/**
 * database.php
 * Connexion PDO — pattern Singleton
 */

/**
 * Retourne une instance PDO unique
 *
 * @return PDO  Instance PDO configurée
 * @throws DatabaseException Si la connexion échoue
 */
function getConnexion(): PDO
{
    static $pdo = null;

    if ($pdo !== null) {
        return $pdo;
    }

    $dsn = sprintf(
        "mysql:host=%s;dbname=%s;charset=%s",
        DB_HOST,
        DB_NAME,
        DB_CHARSET
    );

    try {
        $pdo = new PDO($dsn, DB_USER, DB_PASS, [
            PDO::ATTR_ERRMODE            => PDO::ERRMODE_EXCEPTION,
            PDO::ATTR_DEFAULT_FETCH_MODE => PDO::FETCH_ASSOC,
            PDO::ATTR_EMULATE_PREPARES   => false,
        ]);
    } catch (PDOException $e) {
        // On loggue le vrai message mais on n'expose rien
        error_log("PDO connexion échouée : " . $e->getMessage());
        throw new DatabaseException("Connexion base de données impossible", 500);
    }

    return $pdo;
}
```

### **`src/helpers/crud.php` — version PDO**

```php
<?php
declare(strict_types=1);

/**
 * crud.php
 * Fonctions CRUD génériques — version PDO
 */

/**
 * Vérifie qu'un nom de table est sûr
 * Les noms de tables ne peuvent pas être préparés en SQL
 *
 * @throws ValidationException Si le nom contient des caractères interdits
 */
function validerNomTable(string $table): void
{
    if (!preg_match('/^[a-zA-Z0-9_]+$/', $table)) {
        throw new ValidationException("Nom de table invalide : {$table}");
    }
}

/**
 * Récupère tous les enregistrements d'une table
 *
 * @param string $table   Nom de la table
 * @param string $orderBy Colonne de tri (optionnel)
 * @return array          Tableau de résultats
 */
function fetchAll(string $table, string $orderBy = 'id'): array
{
    validerNomTable($table);
    validerNomTable($orderBy);

    $pdo  = getConnexion();
    $stmt = $pdo->prepare("SELECT * FROM `{$table}` ORDER BY `{$orderBy}` DESC");
    $stmt->execute();

    return $stmt->fetchAll();
}

/**
 * Récupère un enregistrement par son ID
 *
 * @param string $table Nom de la table
 * @param int    $id    ID de l'enregistrement
 * @return array        L'enregistrement trouvé
 * @throws NotFoundException Si introuvable
 */
function fetchById(string $table, int $id): array
{
    validerNomTable($table);

    $pdo  = getConnexion();
    $stmt = $pdo->prepare("SELECT * FROM `{$table}` WHERE id = ?");
    $stmt->execute([$id]);

    $enregistrement = $stmt->fetch();

    if (!$enregistrement) {
        throw new NotFoundException(
            "Enregistrement ID {$id} introuvable dans `{$table}`",
            404
        );
    }

    return $enregistrement;
}

/**
 * Insère un enregistrement
 *
 * @param string $table   Nom de la table
 * @param array  $donnees ['colonne' => 'valeur']
 * @return int            ID inséré
 */
function insert(string $table, array $donnees): int
{
    validerNomTable($table);

    $pdo          = getConnexion();
    $colonnes     = implode(', ', array_map(fn($c) => "`{$c}`", array_keys($donnees)));
    $placeholders = implode(', ', array_fill(0, count($donnees), '?'));

    $stmt = $pdo->prepare("INSERT INTO `{$table}` ({$colonnes}) VALUES ({$placeholders})");
    $stmt->execute(array_values($donnees));

    return (int)$pdo->lastInsertId();
}

/**
 * Met à jour un enregistrement par son ID
 *
 * @param string $table   Nom de la table
 * @param int    $id      ID de l'enregistrement
 * @param array  $donnees Données à mettre à jour
 * @return int            Nombre de lignes affectées
 */
function update(string $table, int $id, array $donnees): int
{
    validerNomTable($table);

    $pdo = getConnexion();

    $set = implode(', ', array_map(
        fn(string $col): string => "`{$col}` = ?",
        array_keys($donnees)
    ));

    $valeurs   = array_values($donnees);
    $valeurs[] = $id;

    $stmt = $pdo->prepare("UPDATE `{$table}` SET {$set} WHERE id = ?");
    $stmt->execute($valeurs);

    return $stmt->rowCount();
}

/**
 * Supprime un enregistrement par son ID
 *
 * @param string $table Nom de la table
 * @param int    $id    ID de l'enregistrement
 * @return int          Nombre de lignes supprimées
 */
function delete(string $table, int $id): int
{
    validerNomTable($table);

    $pdo  = getConnexion();
    $stmt = $pdo->prepare("DELETE FROM `{$table}` WHERE id = ?");
    $stmt->execute([$id]);

    return $stmt->rowCount();
}
```

## 4. Ce que font les seniors 👨‍💻

Les seniors utilisent PDO avec des **paramètres nommés** et organisent leurs requêtes dans des fichiers dédiés par entité — ce qu'on appelle le pattern **Repository** :

```php
<?php
// src/repositories/ArticleRepository.php
declare(strict_types=1);

/**
 * Toutes les requêtes liées aux articles
 * au même endroit — facile à maintenir
 */

/**
 * Récupère les articles publiés avec pagination
 *
 * @param int $page    Numéro de page
 * @param int $parPage Nombre d'articles par page
 * @return array       Articles + métadonnées
 */
function getArticlesPublies(int $page = 1, int $parPage = 10): array
{
    $pdo    = getConnexion();
    $offset = ($page - 1) * $parPage;

    // Compter le total
    $stmt  = $pdo->prepare("SELECT COUNT(*) as total FROM articles WHERE statut = 'publie'");
    $stmt->execute();
    $total = (int)$stmt->fetch()['total'];

    // Récupérer la page
    $stmt = $pdo->prepare("
        SELECT id, titre, auteur, created_at
        FROM articles
        WHERE statut = 'publie'
        ORDER BY created_at DESC
        LIMIT :limite
        OFFSET :offset
    ");

    // Avec les paramètres nommés — bindValue pour les entiers
    $stmt->bindValue(':limite',  $parPage, PDO::PARAM_INT);
    $stmt->bindValue(':offset',  $offset,  PDO::PARAM_INT);
    $stmt->execute();

    return [
        'articles'    => $stmt->fetchAll(),
        'total'       => $total,
        'pages'       => ceil($total / $parPage),
        'page_actuelle' => $page,
    ];
}

/**
 * Recherche des articles par mot-clé
 *
 * @param string $motCle Mot-clé à rechercher
 * @return array         Articles correspondants
 */
function rechercherArticles(string $motCle): array
{
    $pdo  = getConnexion();
    $like = '%' . $motCle . '%';

    $stmt = $pdo->prepare("
        SELECT id, titre, auteur, created_at
        FROM articles
        WHERE statut = 'publie'
        AND (titre LIKE :motcle OR contenu LIKE :motcle)
        ORDER BY created_at DESC
    ");

    $stmt->execute([':motcle' => $like]);

    return $stmt->fetchAll();
}
```

```php
    $like = '%' . $motCle . '%';

    $stmt = $pdo->prepare("
        SELECT *
        FROM articles
        WHERE titre   LIKE :motcle
        OR    contenu LIKE :motcle
        ORDER BY created_at DESC
    ");
```

# Le schéma mental PDO à retenir
```
┌────────────────────────────────────────────────────────────┐
│  MySQLi vs PDO                                             │
│                                                            │
│  MySQLi → mysqli_prepare() + bind_param("ssi", ...)       │
│  PDO    → prepare()        + execute([$a, $b, $c])        │
│                                                            │
│  PDO est plus concis et multi-base de données             │
├────────────────────────────────────────────────────────────┤
│  PARAMÈTRES NOMMÉS vs POSITIONNELS                        │
│                                                            │
│  Positionnels : WHERE id = ?  → execute([$id])            │
│  Nommés       : WHERE id = :id → execute([':id' => $id])  │
│                                                            │
│  Les deux fonctionnent — les nommés sont plus lisibles    │
│  pour les requêtes avec beaucoup de paramètres            │
├────────────────────────────────────────────────────────────┤
│  TRANSACTION — tout ou rien                               │
│                                                            │
│  beginTransaction()                                        │
│  try { ... commit() }                                      │
│  catch { rollBack() }                                      │
└────────────────────────────────────────────────────────────┘
```
